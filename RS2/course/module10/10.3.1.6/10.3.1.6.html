Sauvegarde et restauration par USB

Sauvegarde des configurations à l'aide d'un disque Flash USB

Lorsque vous effectuez une sauvegarde vers un port USB, il est recommandé d'exécuter la commande show file systems pour vérifier que le disque USB est connecté et pour confirmer son nom, comme le montre la figure 1.

Ensuite, utilisez la commande copy run usbflash0:/ pour copier le fichier de configuration sur la clé USB à mémoire Flash. Veillez à utiliser le nom du disque Flash tel qu'il apparait dans le système de fichiers. La barre oblique est facultative et indique le répertoire racine du disque Flash USB.

IOS vous invite à indiquer le nom du fichier. Si le fichier existe déjà sur le périphérique USB, le routeur demande s'il peut le remplacer, comme le montre la figure 2.

Utilisez la commande dir pour afficher le fichier sur le disque USB et la commande more pour voir le contenu, comme l'illustre la figure 3.

Restauration des configurations à l'aide d'un disque Flash USB

Pour copier le fichier, il est nécessaire de modifier le fichier USB R1-Config au moyen d'un éditeur de texte. En supposant que le nom du fichier est R1-Config, utilisez la commande copy usbflash0:/R1-Config running-config pour restaurer la configuration en cours.

La Figure 1 affiche la sortie de la commande « show file systems » sur un routeur. Le système de fichiers « usbflash0: » est mis en surbrillance. La Figure 2 affiche la sortie de la commande « copy running-config usbflash0: » deux fois. Une fois lorsque le fichier de destination n'existe pas et une fois lorsqu'il existe. Si le fichier de destination existe déjà, l'utilisateur est invité à l'écraser. La Figure 3 montre la sortie de la commande « dir usbflash0:/ ». Elle montre également la capacité à afficher le fichier R1-config dans la mémoire Flash via la commande « more usbflash0:/R1-Config ».
