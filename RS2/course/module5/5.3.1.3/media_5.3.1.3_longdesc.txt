La Figure 1 affiche un tableau qui regroupe les commandes de configuration de base pour les interfaces de commutateur.  La CLI commence en mode EXEC privilégié et la première commande est : configure terminal.  Cela fait passer en mode de configuration globale.  La commande suivante est « interface fast Ethernet 0/1 » et l’invite du commutateur devient « config-i.f. » entre parenthèses.  Cela fait passer au mode de configuration pour l’interface spécifiée.  La commande suivante est : duplex full. Cette commande règle l’option duplex sur intégrale.  La commande suivante est : speed 100. Cette commande règle la vitesse sur l’interface à 100 mégabits par seconde.  La configuration en cours est copiée vers le fichier de configuration de démarrage avec la commande : copy running-config startup-config.
