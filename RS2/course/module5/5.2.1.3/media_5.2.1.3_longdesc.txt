La Figure 1 affiche la fenêtre de configuration pour PuTTY. Dans cet exemple, le type de connexion est configuré sur SSH, le port est défini sur 22 et l’adresse IP de l’hôte avec laquelle se connecter est configurée. La Figure 2 affiche la fenêtre de connexion actuelle pour PuTTY. Dans cet exemple, le commutateur nécessite le nom d’utilisateur et le mot de passe. La Figure 3 affiche la sortie de la commande « show ip ssh ».  La sortie répertorie la version de SSH qui s’exécute sur le commutateur, ainsi que d’autres informations de configuration.  La Figure 3 affiche également la sortie de la commande « show ssh ».  Cette sortie répertorie les connexions ssh actuelles qui s’exécutent sur le commutateur.
