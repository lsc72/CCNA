La Figure 1 présente les commandes utilisées pour configurer une description sur une interface.  Les commandes commencent en mode de configuration globale.  La première commande passe au mode de configuration d’interface ; il s'agit de « interface gigabit ethernet 0/0 ».  La commande suivante configure la description ; il s’agit de la description Lien à LAN 1.  Le texte après le mot description devient la description.  La commande suivante dans la figure attribue une adresse IP à l’interface. Il s’agit de l’adresse IP 192.168.10.1 255.255.255.0.  L’interface est démarrée avec la commande « no shutdown ». La Figure 2 configure une description et une adresse IP sur le port Gigabit Ethernet 0/0.  Les commandes commencent en mode de configuration globale.  La première commande passe au mode de configuration d’interface ; il s’agit de « interface gigabit ethernet 0/1 ».  La commande suivante configure la description ; il s’agit de la description Lien à LAN 2.  La commande suivante dans la figure attribue une adresse IP à l’interface. Il s’agit de l’adresse IP 192.168.11.1 255.255.255.0.  L’interface est démarrée avec la commande « no shutdown ». La Figure 3 configure une interface série et commence en mode de configuration globale.  La première commande passe au mode de configuration d’interface, il s’agit de « interface serial 0/0/0 ».  La commande suivante configure la description ; il s’agit de la description Lien à R2.  La commande suivante dans la figure attribue une adresse IP à l’interface. Il s’agit de l’adresse IP 209.165.200.255 255.255.255.252.  La commande suivante configure une fréquence d’horloge sur l’interface série. La commande est « clock rate 128,000 ».  L’interface est démarrée avec la commande « no shutdown ». La Figure 5 est une activité interactive qui permet à l’étudiant de configurer une adresse IP, d’ajouter une description et d’activer une interface sur un routeur.
