La Figure 1 affiche un réseau constitué d'un routeur avec une interface série connectée à Internet et deux interfaces Gigabit Ethernet locales configurées sur un réseau local. Tous les réseaux sont associés à des adresses IPv6. La Figure 2 illustre l'applet de Windows pour attribuer de manière statique une adresse IPv6 sur un ordinateur. La Figure 3 illustre le routeur I.O.S. Commandes de l'interface de ligne (CLI) pour configurer le protocole IPv6 sur une interface.  Les commandes commencent en mode de configuration globale.  La première commande passe au mode de configuration d’interface ; il s’agit de « interface gigabit ethernet 0/0 ».  La commande suivante configure la description ; il s’agit de la description Lien à LAN 1.  La commande suivante dans la figure attribue une adresse IP à l’interface. Il s’agit de l’adresse IPv6 2001:DB8:ACAD:1::1/64.  L’interface est démarrée avec la commande « no shutdown ». La Figure 4 configure l’interface suivante et commence en mode de configuration globale.  La première commande passe au mode de configuration d’interface ; il s’agit de « interface gigabit ethernet 0/1 ».  La commande suivante configure la description ; il s’agit de la description Lien à LAN 2.  La commande suivante dans la figure attribue une adresse IP à l’interface. Il s’agit de l’adresse IPv6 2001:DB8:ACAD:1::2/64.  L’interface est démarrée avec la commande « no shutdown ». La Figure 5 configure l’interface suivante et commence en mode de configuration globale.  La commande est « interface serial 0/0/0 ».  La commande suivante configure la description ; il s’agit de la description Lien à R2.  La commande suivante dans la figure attribue une adresse IP à l’interface. Il s’agit de l’adresse IP 2001:DB8:ACAD:1::3/64.  La commande suivante configure une fréquence d’horloge sur l’interface série. La commande est « clock rate 128,000 ».  L’interface est démarrée avec la commande « no shutdown ». La Figure 6 est une activité interactive qui permet à l'étudiant de configurer et d'activer trois interfaces sur un routeur.
