La Figure 1 affiche la commande pour ajouter une description et pour attribuer une adresse IP à une interface.  La figure affiche l’interface de ligne de commande pour un routeur et commence en mode de configuration globale.  La première commande passe à l’interface souhaitée, il s’agit de la commande : « interface gigabit Ethernet 0/0 ».  La commande suivante inscrit une description dans le fichier de configuration pour l’interface, et cette commande-ci est une description de la Liaison vers LAN 1.  La commande suivante attribue l’adresse IP et le masque de sous-réseau et cette adresse IP est 192.168.10.1 255.255.255.0.  L’interface doit à présent être démarrée avec la commande « no shutdown ».  L’interface a ensuite été basculée à l’état ouvert. La Figure 2 configure l’interface suivante.  La première commande passe à l’interface, il s’agit de la commande « interface gigabit ethernet 0/1 ».  La commande suivante inscrit une description dans le fichier de configuration pour l’interface, et cette commande-ci est une description de la Liaison vers LAN 2.  La commande suivante attribue l’adresse IP et le masque de sous-réseau et cette adresse IP est 192.168.11.1 255.255.255.0.  L’interface doit à présent être démarrée avec la commande « no shutdown ».  L’interface a été basculée à l’état ouvert. La Figure 3 configure l’interface suivante.  La première commande passe à l’interface, il s’agit de la commande « interface serial 0/0/0 ».  La commande suivante inscrit une description dans le fichier de configuration pour l’interface et cette commande-ci est une description de la Liaison vers R1.  La commande suivante attribue l’adresse IP et le masque de sous-réseau et cette adresse IP est 209.165.200.225 255.255.255.0.  Ensuite, une fréquence d’horloge doit être attribuée à la liaison série, il s’agit de la commande « clock rate 128000 ».  L’interface doit à présent être démarrée avec la commande « no shutdown ».  L’interface a été basculée à l’état ouvert. La Figure 4 affiche la sortie de la commande « show i.p. route route pipe Gateway ».  Elle présente la table de routage et applique le filtre pour afficher tous les éléments du fichier qui résultent après la saisie du terme Gateway, incluant le mot clé Gateway.  La table présente les six entrées prévues suite aux trois interfaces configurées dans les trois figures précédentes. La Figure 5 est un élément multimédia interactif qui permet à l’apprenant de configurer trois interfaces sur un routeur pour qu’il puisse se connecter à ce routeur dans les quatre figures précédentes.
