Exemples de mot-clé de masque générique

L'Exemple 1 de la figure indique comment utiliser le mot-clé any pour remplacer l'adresse IPv4 0.0.0.0, avec un masque générique 255.255.255.255.

L'exemple 2 montre comment utiliser le mot-clé host pour remplacer le masque générique lors de l'identification d'un hôte unique.

Remarque : la syntaxe pour configurer les listes de contrôle d’accès IPv4 standard est présentée plus loin dans ce chapitre.

La figure présente deux exemples d’utilisation des commandes de CLI pour la création d’une liste d’accès autorisant toute adresse. Les exemples commencent en mode de configuration globale. Le premier exemple répertorie la méthode à l'aide des masques génériques. La commande est « access-list1 permit 0.0.0.0 255.255.255.255 ». Le prochain exemple utilise le mot clé « any », et la commande est « access-list 1 permit any ». Le deuxième exemple est une liste de contrôle d'accès qui spécifie un hôte particulier. La commande est « access-list 1 permit 192.168.10.10 0.0.0.0 ». La commande suivante de l'Exemple 2 utilise le mot clé « host ». La commande est « access-list 1 permit host 192.168.10.10 ».
