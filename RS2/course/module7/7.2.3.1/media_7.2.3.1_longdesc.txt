La Figure 1 affiche un routeur avec un segment de réseau local sur chacune des deux interfaces Gigabit Ethernet.  Chaque segment se compose d'un commutateur et d'un ordinateur.  Le routeur dispose des ports VTY 0 à 4 configurés pour l'accès.  La figure illustre les commandes requises pour créer une liste de contrôle d'accès qui limite les connexions VTY.  La figure débute en mode de configuration globale puis se déplace aux lignes VTY avec la commande « line vty 0 4 ».  La commande suivante configure le routeur pour utiliser une base de données locale pour l'accès avec la commande « login local ».  La ligne suivante définit le type de connexion à autoriser. La commande est « transport input ssh ».  Ensuite intervient l'application d'une liste de contrôle d'accès sur les lignes avec la commande « access-class 21 in ». La figure passe du mode de configuration de ligne au mode de configuration globale avec la commande « exit ».  La liste de contrôle d'accès est créée avec la commande « access-list 21 permit 192.168.10.0 0.0.0.255 » et la commande « access-list 21 deny any ». La Figure 2 est un élément multimédia interactif qui permet à l’étudiant de configurer les lignes VTY pour accepter SSH sur un routeur.
