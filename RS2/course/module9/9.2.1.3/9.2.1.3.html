Vérification de la NAT statique

La commande show ip nat translations permet de vérifier le fonctionnement de la NAT. Elle affiche les traductions NAT actives. Les traductions statiques, contrairement aux traductions dynamiques, se trouvent toujours dans la table NAT. La Figure 1 présente le résultat de cette commande dans l'exemple de configuration précédent. Comme l'exemple est une configuration NAT statique, la traduction est toujours présente dans la table NAT, qu'il y ait ou non des communications actives. Si la commande est émise pendant une session active, le résultat indique également l'adresse du périphérique externe comme à la Figure 1.

La commande show ip nat statistics est également utile. Comme le montre la Figure 2, la commande show ip nat statistics affiche des informations sur le nombre total de traductions actives, les paramètres de configuration NAT, le nombre d’adresses dans le pool et le nombre d’adresses allouées.

Pour vérifier que la traduction NAT fonctionne, il est préférable d'effacer les statistiques des traductions passées à l'aide de la commande clear ip nat statistics avant d'effectuer le test.

Avant toute communication avec le serveur Web, la commande show ip nat statistics n'affiche aucune occurrence. Lorsque le client établit une session avec le serveur Web, la commande show ip nat statistics montre plus de résultats (cinq occurrences) sur l’interface interne (Serial0/0/0). Cela permet de vérifier que la traduction NAT statique a lieu sur R2.

La Figure 1 affiche le résultat de la commande « show i. p. nat translations » sans session active. Les adresses externes ne sont pas indiquées. La figure présente également le résultat de la commande « show i. p. nat translations » avec une session active. Les adresses externes sont présentes. Cela montre que les traductions NAT sont toujours présentes lors d’une traduction d’adresses réseau statique. La Figure 2 affiche le résultat de la commande « clear i. p. nat statistics » suivie de la commande « show i. p. nat statistics » qui n’affiche aucune occurrence. Lorsqu’un PC établit une session avec le serveur Web, le résultat de la commande « show i. p. nat statistics » contient cinq occurrences.
