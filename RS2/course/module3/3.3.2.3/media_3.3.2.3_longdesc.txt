La Figure 1 répertorie trois types de routes de niveau 1 et présente un exemple de chacun de ces types de routes.  Le premier exemple est 192.168.1.0/24, le second est 192.168.0.0/16 et le dernier est une route par défaut de 0.0.0.0/0.  Ce sont toutes des routes finales dans une table de routage lorsque l’adresse IP du tronçon suivant ou l’interface de sortie est répertoriée dans la table de routage. La Figure 2 affiche le résultat de la commande : show i.p. route pipe begin Gateway.  Elle présente la table de routage pour le routeur R-1 et met en surbrillance deux routes de niveau 2. L’une des routes est la route statique par défaut et l’autre est 192.168.0.0/16 accessible via 209.165.200.226.
