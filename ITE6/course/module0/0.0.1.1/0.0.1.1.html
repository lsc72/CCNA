Accueil

Bienvenue dans le cours IT Essentials (Information Technology Essentials) L'objectif de ce cours est de vous présenter le matériel informatique et les logiciels, ainsi que les systèmes d'exploitation, les concepts liés aux réseaux, les appareils mobiles, la sécurité informatique et le dépannage. Ces supports de cours en ligne vous aideront à développer les compétences nécessaires pour travailler en tant que technicien dans le domaine de l'informatique. Les connaissances visées sont présentées au début de chaque chapitre.

Vous pouvez utiliser un smartphone, une tablette, un ordinateur portable ou un ordinateur de bureau pour accéder au cours, participer aux discussions avec votre instructeur, consulter vos notes, lire ou réviser des textes et tester vos connaissances à l'aide de contenus interactifs. Vous pouvez également répondre aux questionnaires ou passer les examens sur votre appareil mobile. Vous pouvez télécharger l'application Packet Tracer pour mobile pour réaliser les exercices de ce cours.

Remarque : sur vote mobile, vous ne pourrez pas passer les examens et questionnaires en ligne qui utilisent Packet Tracer.
