ACPI standard: S0, description: The computer is on and the CPU is running at full power. ACPI standard: S1, description: The CPU is not executing instructions, however, the CPU and RAM are still receiving power. ACPI standard: S2, description: The CPU is off, but the RAM is refreshed to maintain the contents. ACPI standard: S3, description: The CPU is off, and the RAM is set to a slow refresh rate. ACPI standard: S4, description: The CPU and RAM are off. The contents of RAM have been saved to a temporary file on the hard disk. ACPI standard: S5, description: The computer is off and any content that has not been saved will be lost.
