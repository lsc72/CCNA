Paramètres proxy

Les serveurs proxy sont des serveurs mandataires et agissent à la place des ordinateurs. Les serveurs proxy servent généralement de stockage ou de cache pour les pages Web fréquemment consultées sur les périphériques du réseau interne. Ainsi, le serveur proxy illustré ci-contre stocke les pages Web de www.cisco.com. Lorsqu'un hôte interne envoie une requête HTTP GET à www.cisco.com, le serveur proxy procède comme suit :

1. Il intercepte les requêtes.

2. Il vérifie si le contenu du site Web a changé.

3. Dans la négative, le serveur proxy répond à l'hôte avec la page Web.

En outre, un serveur proxy peut masquer efficacement les adresses IP des hôtes internes, car toutes les requêtes sortantes d'Internet ont comme adresse source l'adresse IP du serveur proxy.
