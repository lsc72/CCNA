The figure on this page is a topology that shows a sender connected to Router1, which is connected to the internet. A recipient is connected to Router2, which is also connected to the internet. The mail server for ISP A is connected to a router, which is connected to the internet. The mail server for ISP is connected to a router, which is connected to the internet. The recipient is sending an email message to, ‘recipient@cisco.com’. The email message is forwarded from the sender to the web server at ISP A using SMTP. The web server at ISP A then forwards the email message to the web server at ISP B using SMTP. The web server at ISP B then forwards the email message to the recipient using IMAP or POP3.
