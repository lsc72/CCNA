Public, this cloud type is made available to the public by a service provider who hosts the cloud infrastructure. IaaS, the cloud provider supplies the network equipment, virtualized network services, and the supporting network infrastructure. Community, this cloud type is shared among several organizations. Private, this cloud infrastructure is managed and used by a particular organization. SaaS, The cloud provider manages and delivers the end-user applications over the internet. Hybrid, this cloud infrastructure is a combination of two or more cloud types: public, private, or community. PaaS, the cloud provider manages the infrastructure, such as the hardware and the operating system. ITaaS, an operating model for delivering IT support for SaaS, PaaS, and IaaS.
