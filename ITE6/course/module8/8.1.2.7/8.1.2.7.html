Test de connectivité avec l'interface graphique utilisateur Windows

Lorsque tous les périphériques sont connectés et les voyants allumés, testez la connectivité du réseau. La méthode la plus simple pour tester une connexion Internet est d'ouvrir le navigateur et d'essayer d'accéder à une page web. Pour dépanner une connexion sans fil, vous pouvez utiliser l'interface graphique utilisateur (GUI) ou l'interface en ligne de commande (CLI) de Windows.

Pour contrôler une connexion sans fil sous Windows Vista, procédez comme suit :

Démarrer > Panneau de configuration > Réseau et Internet > Centre Réseau et partage > Gérer les connexions réseau.

Pour contrôler une connexion sans fil ou filaire sous Windows 7, 8 ou 8.1, procédez comme suit :

Démarrer > Panneau de configuration > Réseau et Internet > Centre Réseau et partage > Modifier les paramètres de la carte.

Sous Windows 7 et Windows Vista, les connexions réseau filaires sont généralement appelées des « Connexions au réseau local », voir l'illustration 1. Sous Windows 8 et Windows 8.1, les connexions réseau filaires sont appelées « Ethernet », voir l'illustration 2.

Double-cliquez sur la connexion réseau pour afficher l'écran d'état de Windows 7, voir l'illustration 3. L'écran d'état indique si l'ordinateur est connecté à Internet, ainsi que la durée de la connexion. Il indique également le nombre d'octets envoyés et reçus.

Dans toutes les versions de Windows, cliquez sur le bouton Détails pour afficher les informations d'adressage IP, le masque de sous-réseau, la passerelle par défaut, l'adresse MAC et d'autres informations, voir l'illustration 4. Si la connexion ne fonctionne pas correctement, fermez la fenêtre Détails et cliquez sur Diagnostiquer pour réinitialiser les informations de connexion et tenter d'établir une nouvelle connexion.
