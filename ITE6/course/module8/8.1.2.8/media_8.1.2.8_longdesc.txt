The figure on this page is a table that lists the following ipconfig command options. Command Option: /all, Purpose: Displays full configuration information of all network adapters. Command Option: /release, Purpose: Releases the IP address of a network adapter. Command Option: /renew, Purpose: Renews the IP address of a network adapter. Command Option: /flushdns, Purpose: Empties the cache that stores DNS information. Command Option: /registerdns, Purpose: Refreshes the DHCP lease and re-registers the adaptor with DNS. Command Option: /displaydns, Purpose: Shows DNS information in the Cache.
