Configuration d'une carte réseau

Une fois le pilote de la carte réseau installé, configurez les paramètres d'adresse IP. Une adresse IP peut être affectée à un ordinateur de deux manières différentes :

  • Manuellement – Une configuration IP est attribuée à l'hôte de manière statique.

  • Dynamiquement – L'hôte demande la configuration de son adresse IP à un serveur DHCP.

Pour configurer manuellement les paramètres IP sous Windows 8 et Windows 8.1, procédez comme suit :

Panneau de configuration > Réseau et Internet > Centre Réseau et partage > Modifier les paramètres de la carte > cliquez avec le bouton droit sur Ethernet. La fenêtre des propriétés d'Ethernet s'ouvre.

Sous Windows 7 et Windows Vista, sélectionnez :

Démarrer > Panneau de configuration > Centre Réseau et partage > Modifier les paramètres de la carte > cliquez avec le bouton droit sur Connexion au réseau local. La fenêtre Propriétés de la connexion au réseau local (illustration 1) s'ouvre.

Pour configurer les paramètres IPv4, cliquez sur Protocole Internet version 4 (TCP/IPv4) > Propriétés. La fenêtre Protocole Internet version 4 (TCP/IPv4) (illustration 2) s'ouvre. Notez que l'option par défaut est d'obtenir les paramètres IP automatiquement auprès du serveur DHCP. Pour configurer les paramètres manuellement, cliquez sur Utiliser l'adresse IP suivante comme indiqué à l'illustration 3. Entrez ensuite l'adresse IPv4, le masque de sous-réseau et l'adresse de la passerelle par défaut, puis cliquez sur OK > OK.

Pour configurer les paramètres IPv6, cliquez sur Protocole Internet version 6 (TCP/IPv6 ) > Propriétés. La fenêtre Protocole Internet version 6 (TCP/IPv6), indiquée à l'illustration 4, s'ouvre. Cliquez sur Utiliser l'adresse IPv6 suivante, comme le montre l'illustration 5. Entrez ensuite l'adresse IPv6, la longueur de préfixe et l'adresse de la passerelle par défaut, puis cliquez sur OK > OK.

Remarque : la plupart des ordinateurs actuels disposent d'une carte réseau intégrée. Si vous installez une nouvelle carte réseau, il est conseillé de désactiver la carte réseau intégrée dans les paramètres du BIOS.
