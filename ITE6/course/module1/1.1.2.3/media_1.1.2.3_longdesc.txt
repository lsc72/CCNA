The figure on this page is a table listing the following descriptions for multicore processors. Number of Cores: Single core CPU, Description: One core inside a single CPU that handles all the processing. A motherboard may have sockets for more than one single processor, providing the ability to build a powerful, multiprocessor compute. Number of Cores: Dual core CPU, Description: Two cores inside a single CPU in which both cores can process information at the same time. Number of Cores: Triple core CPU, Description: Three cores inside a single CPU. This is a quad core processor with one of the cores disabled. Number of Cores: Quad core CPU, Description: Four cores inside a single CPU. Number of Cores: Hexa-core CPU, Description: Six cores inside a single CPU. Number of Cores: Octa-core CPU, Description: Eight cores inside a single CPU.
