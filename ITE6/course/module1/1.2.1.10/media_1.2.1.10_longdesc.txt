Figure 1 on this page shows an image of an optical drive. The figure also lists the following factors to be considered before purchasing a new optical drive: Connector type, reading capability, writing capability, optical media type. Figure 2 on this page is a table summarising the following optical drive capabilities: CD-ROM: Read CD. CD-RW: Read CD, write CD. DVD-ROM: Read CD, read DVD. DVD-RW: Read CD, write CD, read DVD, write DVD. BD-ROM: Read CD, read DVD, read Blu-ray. BD-R: read CD, write CD, read DVD, write DVD, read Blu-ray, write Blu-ray. BD-RE: read CD, write CD, read DVD, write DVD, read Blu-ray, write Blu-ray, rewrite Blu-ray.
