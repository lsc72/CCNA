Mise à jour et installation des pilotes de l'imprimante

Une fois que vous avez connecté les câbles d'alimentation et de données à l'imprimante, le système d'exploitation détecte cette imprimante et installe le pilote.

Le pilote d'impression est un logiciel qui permet à un ordinateur et à une imprimante de communiquer. Le pilote met également à la disposition de l'utilisateur une interface lui permettant de configurer les options d'impression. Chaque modèle d'imprimante est associé à un pilote unique. Les fabricants d'imprimantes mettent fréquemment à jour leurs pilotes pour améliorer les performances des imprimantes, ajouter des options ou résoudre des problèmes. Vous pouvez télécharger les pilotes d'impression les plus récents depuis le site Web du fabricant de l'imprimante.

Pour installer un pilote d'impression, procédez comme suit :

Étape 1. Déterminer si un pilote plus récent est disponible. La plupart des sites Web des fabricants comportent un lien vers une page qui propose des pilotes et une assistance technique. Assurez-vous que le pilote est compatible avec l'ordinateur et le système d'exploitation que vous utilisez.

Étape 2. Télécharger les fichiers du pilote sur votre ordinateur. La plupart des fichiers de pilotes sont compressés ou zippés. Téléchargez le fichier dans un dossier et décompressez le contenu. Enregistrez les instructions ou la documentation dans un dossier séparé sur votre ordinateur.

Étape 3. Installez le pilote automatiquement ou manuellement. La plupart des pilotes d'impression disposent d'un fichier d'installation qui recherche automatiquement les anciens pilotes dans le système et les remplace par les nouveaux. Si aucun fichier d'installation n'est disponible, conformez-vous aux instructions fournies par le fabricant.

Étape 4. testez le nouveau pilote d'imprimante. Effectuez plusieurs tests pour vérifier que l'imprimante fonctionne correctement. Utilisez des applications variées pour imprimer différents types de documents. Modifiez et testez chaque option d'impression.

La page de test imprimée doit contenir du texte lisible. Si le texte est illisible, le problème peut provenir d'un pilote inadapté ou de l'utilisation du mauvais langage de description de page.
