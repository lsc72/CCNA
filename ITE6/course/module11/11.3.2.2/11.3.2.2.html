Serveurs d'impression logiciels

Dans la section précédente, vous avez appris comment un ordinateur Windows peut partager une imprimante connectée avec d'autres ordinateurs Windows sur le réseau. Mais que se passe-t-il si l'ordinateur partageant l'imprimante exécute un autre système d'exploitation, comme Mac OS X ? Dans cette situation, vous pouvez utiliser un logiciel de serveur d'impression.

Il peut s'agir par exemple du serveur d'impression gratuit Bonjour d'Apple, qui est un service intégré à Mac OS X. Il est automatiquement installé sur les ordinateurs Windows si vous installez le navigateur Apple Safari. Vous pouvez également télécharger gratuitement le serveur d'impression Bonjour pour Windows sur le site Web d'Apple.

Une fois téléchargé et installé, le serveur d'impression Bonjour s'exécute en arrière-plan et détecte automatiquement toutes les imprimantes compatibles qui sont connectées au réseau. Ouvrez l'assistant d'imprimante Bonjour pour configurer l'ordinateur Windows de sorte qu'il utilise une imprimante, comme illustré ci-contre.

Le partage d'une imprimante à partir d'un ordinateur a également des inconvénients. L'ordinateur partageant l'imprimante utilise ses propres ressources pour gérer les travaux d'impression qui arrivent. Si un utilisateur travaille sur l'ordinateur hôte pendant qu'un autre imprime sur le réseau, il peut constater une baisse des performances. De plus, l'imprimante n'est pas disponible si l'utilisateur redémarre ou éteint l'ordinateur connecté à l'imprimante partagée.
