Installation d'une imprimante

Lorsque vous achetez une imprimante, les informations d'installation et de configuration sont généralement fournies par le fabricant. Un CD d'installation, qui contient les pilotes, les manuels et le logiciel de diagnostic, est fourni avec l'imprimante. Si aucun CD n'est fourni, vous pouvez télécharger ces outils sur le site Web du fabricant.

Bien que les opérations de connexion et de configuration varient d'un type d'imprimante à l'autre, certaines procédures peuvent être appliquées à toutes les imprimantes. Avant d'installer une imprimante, veillez à retirer tous les éléments de l'emballage. Retirez tout ce qui empêche les pièces mobiles de se déplacer pendant le transport. Conservez les emballages d'origine au cas où vous devriez renvoyer l'imprimante au fabricant pour des réparations sous garantie.

Remarque : avant de connecter l'imprimante à l'ordinateur, lisez les instructions d'installation. Dans certains cas, le pilote de l'imprimante doit être installé avant que l'imprimante ne soit branchée sur l'ordinateur.

Si l'imprimante dispose d'un port USB, FireWire ou parallèle, connectez le câble correspondant au port de l'imprimante. Connectez l'autre extrémité du câble de données au port correspondant situé à l'arrière de l'ordinateur. Si vous installez une imprimante réseau, connectez le câble réseau au port réseau.

Une fois le câble de données correctement connecté, branchez le câble d'alimentation sur l'imprimante. Branchez l'autre extrémité de ce câble sur une prise électrique disponible. Lorsque vous mettez le périphérique sous tension, l'ordinateur tente de déterminer le pilote correspondant à installer.
