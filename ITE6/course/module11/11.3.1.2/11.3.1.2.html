Connexion à une imprimante partagée

Pour se connecter à une imprimante en réseau depuis un ordinateur sous Windows 7, procédez comme suit :

Étape 1. Sélectionnez Périphériques et imprimantes > Ajouter une imprimante.

Étape 2. L'assistant Ajouter une imprimante s'affiche.

Étape 3. Sélectionnez Ajouter une imprimante réseau, sans fil ou Bluetooth, voir l'illustration 1.

Étape 4. Une liste d'imprimantes partagées s'affiche. Si l'imprimante n'apparaît pas dans la liste, sélectionnez L'imprimante que je recherche n'est pas dans la liste.

Étape 5. Après avoir sélectionné l'imprimante, cliquez sur Suivant.

Étape 6. Un port d'imprimante virtuel est créé et affiché dans la fenêtre Ajouter une imprimante. Les pilotes d'impression nécessaires sont téléchargés depuis le serveur d'impression et installés sur l'ordinateur. L'Assistant termine alors l'installation.

L'imprimante ne figure pas dans la liste

Dans toutes les versions de Windows, si vous cliquez sur L'imprimante que je veux n'est pas répertoriée, vous disposez des options suivantes pour rechercher le chemin vers l'imprimante réseau, voir l'illustration 2 :

  • Rechercher une imprimante sur le réseau

  • Entrer le chemin exact de l'imprimante sur le réseau

  • Entrer l'adresse IP ou le nom de l'imprimante sur le réseau

Connectez-vous à netacad.com avant d'accéder à ce cours.
