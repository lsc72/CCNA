Page de test de l'imprimante

Après avoir installé une imprimante, imprimez une page de test pour vous assurer que celle-ci fonctionne correctement. La page de test permet de vérifier que le pilote est bien installé, qu'il fonctionne correctement et que l'imprimante et l'ordinateur peuvent communiquer.

Pour imprimer manuellement une page de test sous Windows Vista, sélectionnez :

Panneau de configuration > Imprimantes pour afficher la fenêtre Imprimantes.

Cliquez avec le bouton droit sur l'imprimante souhaitée, puis suivez le chemin, comme illustré ci-contre :

Propriétés > onglet Général > Imprimer une page de test

Une boîte de dialogue s'ouvre et demande si la page a été imprimée correctement. Si la page n'a pas été imprimée, des fichiers d'aide intégrés vous aideront à résoudre le problème.

Pour imprimer manuellement une page de test sous Windows 7, sélectionnez :

Périphériques et imprimantes pour afficher le panneau de configuration Périphériques et imprimantes

Cliquez avec le bouton droit de la souris sur l'imprimante souhaitée, puis suivez le chemin :

Propriétés de l'imprimante > onglet Général > Imprimer une page de test

Pour imprimer manuellement une page de test sous Windows 8.0 ou 8.1, sélectionnez :

Panneau de configuration > Périphériques et imprimantes pour afficher le panneau de configuration Périphériques et imprimantes

Cliquez avec le bouton droit de la souris sur l'imprimante souhaitée, puis suivez le chemin :

Propriétés de l'imprimante > onglet Général > Imprimer une page de test

Impression depuis une application

Vous pouvez tester l'imprimante en imprimant une page de test depuis une application telle que le Bloc-notes ou WordPad. Pour accéder au Bloc-notes Sous Windows 7 et Windows Vista, suivez le chemin suivant :

Démarrer > Tous les programmes > Accessoires > Bloc-notes

Pour ouvrir le Bloc-notes sous Windows 8.0 et 8.1, dans l'écran d'accueil, tapez Bloc-notes, puis cliquez sur Bloc-notes pour l'ouvrir.

Tapez du texte dans le document. Imprimez le document en sélectionnant :

Fichier > Imprimer

Test de l'imprimante depuis le panneau de commande de l'imprimante

La plupart des imprimantes sont dotées d'un panneau avant comportant des commandes qui vous permettent d'imprimer des pages de test. Cela vous permet de vérifier que l'imprimante fonctionne en dehors du réseau ou de l'ordinateur. Consultez le site Web ou la documentation du fabricant de l'imprimante pour savoir comment imprimer une page de test depuis le panneau avant.
