Configuration du partage d'imprimantes

Windows permet aux utilisateurs de partager leurs imprimantes avec d'autres utilisateurs du réseau.

Sous Windows 7, pour configurer l'ordinateur auquel est reliée l'imprimante afin qu'il accepte les travaux d'impression provenant d'autres ordinateurs du réseau, procédez comme suit :

Étape 1. Cliquez sur Panneau de configuration > Centre réseau et partage > Modifier les paramètres de partage avancés, comme illustré ci-contre.

Étape 2. Développez la liste des réseaux pour afficher les profils.

Étape 3. Si le partage d'imprimantes est désactivé, sous Partage de fichiers et d'imprimantes, sélectionnez Activer le partage de fichiers et d'imprimantes, puis cliquez sur Enregistrer les modifications.

Sous Windows Vista, pour configurer l'ordinateur auquel est reliée l'imprimante afin qu'il accepte les travaux d'impression provenant d'autres ordinateurs du réseau, procédez comme suit :

Étape 1. Cliquez sur Panneau de configuration > Imprimantes.

Étape 2. Cliquez avec le bouton droit sur l'imprimante à partager et choisissez Partage. La boîte de dialogue Propriétés de l'imprimante s'ouvre.

Étape 3. Sélectionnez Partager cette imprimante et entrez le nom de l'imprimante partagée souhaité. Les autres utilisateurs pourront voir ce nom.

Étape 4. Vérifiez que le partage a réussi. Dans la fenêtre Imprimantes, vérifiez qu'une icône de partage apparaît sous l'icône de l'imprimante. Celle-ci indique qu'une ressource est partagée.

Sous Windows 8.0 et 8.1, pour configurer l'ordinateur auquel est reliée l'imprimante afin qu'il accepte les travaux d'impression provenant d'autres ordinateurs du réseau, procédez comme suit :

Étape 1. Cliquez sur Panneau de configuration > Centre réseau et partage > Modifier les paramètres de partage avancés.

Étape 2. Développez la liste des réseaux pour afficher les profils.

Étape 3. Si le partage d'imprimantes est désactivé, sous Partage de fichiers et d'imprimantes, sélectionnez Activer le partage de fichiers et d'imprimantes, puis cliquez sur Enregistrer les modifications.

D'autres utilisateurs, qui peuvent désormais se connecter à l'imprimante partagée, risquent de ne pas disposer des pilotes requis. Ces utilisateurs peuvent également utiliser des systèmes d'exploitation différents de celui de l'ordinateur qui héberge l'imprimante partagée. Windows peut télécharger automatiquement les pilotes appropriés. Cliquez sur le bouton Pilotes supplémentaires pour sélectionner les systèmes d'exploitation utilisés par les autres utilisateurs. Lorsque vous fermez cette boîte de dialogue en cliquant sur OK, Windows demande à obtenir ces pilotes supplémentaires. Si les autres utilisateurs utilisent le même système d'exploitation Windows, vous n'avez pas besoin de cliquer sur le bouton Pilotes supplémentaires.
