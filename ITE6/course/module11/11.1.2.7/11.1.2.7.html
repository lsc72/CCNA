Imprimantes virtuelles

L'impression virtuelle n'envoie pas les travaux d'impression à une imprimante sur le réseau local. Au lieu de cela, le logiciel d'impression envoie la tâche à un fichier ou bien il transmet les informations à une destination distante dans le cloud pour impression.

Les méthodes standard pour envoyer un travail d'impression à un fichier sont les suivantes :

  • Imprimer dans un fichier : à l'origine, la commande d'impression dans un fichier enregistrait vos données dans un fichier portant l'extension .prn. Ce fichier .prn pouvait ensuite être rapidement imprimé à tout moment sans avoir à ouvrir le document original. La fonction Imprimer dans un fichier peut désormais utiliser d'autres formats, voir l'illustration 1.

  • Imprimer dans un PDF : le format PDF (Adobe Portable Document Format) a été publié en tant que norme ouverte en 2008.

  • Imprimer dans un XPS : créé par Microsoft dans Windows Vista, le format XPS (XML Paper Specification) devait constituer une alternative au format PDF.

  • Imprimer dans une image : pour empêcher que d'autres personnes puissent copier facilement le contenu d'un document, vous pouvez choisir d'imprimer dans un format de fichier image, par exemple JPG ou TIFF.

L'impression cloud envoie le travail d'impression vers une imprimante à distance, voir l'illustration 2. L'imprimante peut se trouver n'importe où sur le réseau de votre entreprise. Certaines imprimeries fournissent des logiciels que vous pouvez installer pour envoyer les travaux d'impression à leur agence la plus proche.

Google Cloud Print est un autre exemple d'impression cloud, qui vous permet de connecter votre imprimante au Web. Une fois qu'elle est connectée, vous pouvez envoyer des travaux d'impression vers l'imprimante où que vous vous trouviez.
