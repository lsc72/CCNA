Options d'impression générales et options spécifiques du document

Certaines imprimantes proposent des commandes permettant de sélectionner des options. D'autres imprimantes utilisent les options du pilote d'impression. Ces options peuvent être paramétrées pour s'appliquer à un seul document ou pour toutes les impressions.

Options générales

Les options générales sont les options d'impression définies pour tous les documents. À chaque impression d'un document, les options générales sont utilisées, sauf si d'autres options (propres au document à imprimer) sont sélectionnées.

Pour modifier les options générales d'une imprimante sous Windows 8 ou Windows 7, sélectionnez :

Panneau de configuration > Périphériques et imprimantes > clic droit sur l'imprimante

Pour définir une imprimante par défaut, cliquez avec le bouton droit sur l'imprimante et sélectionnez Définir comme imprimante par défaut (voir l'illustration).

Pour modifier les options générales d'une imprimante sous Windows Vista, sélectionnez :

Panneau de configuration > Imprimantes et télécopieurs > cliquez avec le bouton droit sur l'imprimante

Pour définir une imprimante par défaut, cliquez avec le bouton droit sur l'imprimante et sélectionnez Définir comme imprimante par défaut.

Remarque : en fonction du pilote installé, l'option Définir comme imprimante par défaut peut être disponible ou non. Si l'option n'apparaît pas, double-cliquez sur l'imprimante pour ouvrir la fenêtre d'état du document, puis sélectionnez Imprimante > Définir comme imprimante par défaut.

Options pour un document spécifique

Les lettres, les tableaux et les photos numériques font partie des types de documents pouvant nécessiter des paramètres d'impression spéciaux. Vous pouvez changer ces paramètres pour un seul document en accédant aux paramètres d'impression du document.
