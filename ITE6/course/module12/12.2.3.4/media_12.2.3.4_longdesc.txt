The figure on this page is a table that lists the following permissions that are available for files and folders. Permission Level: Full Control, Description: User can see the contents of a file or folder, change and delete existing files and folders, and run programs in a folder. Permission Level: Modify, Description: Users can change and delete existing files and folders, but cannot create new ones. Permission Level: Read and Execute, Description: Users can see the contents of existing files and folders and can run programs in a folder. Permission Level: Read, Description: Users can see the contents of a folder and open files and folders. Permission Level: Write, Description: Users can create new files and folders and make changes to existing files and folders.
