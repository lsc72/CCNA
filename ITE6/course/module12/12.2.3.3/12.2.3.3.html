Sauvegardes des données

Des données peuvent être volées ou endommagées (par exemple en cas de panne ou de sinistre). Il est donc important d'effectuer régulièrement une sauvegarde des données.

Une sauvegarde de données stocke une copie des informations de l'ordinateur sur un support amovible qui peut être conservé en lieu sûr. C'est l'un des moyens de protection les plus efficaces contre la perte de données. En cas de panne matérielle de l'ordinateur, vous pouvez restaurer les données à partir de la sauvegarde.

La sauvegarde des données doit donc être effectuée régulièrement. Elle doit faire partie du plan de sécurité. Les sauvegardes sont généralement stockées en dehors des installations, afin de protéger le support de sauvegarde en cas de sinistre dans le bâtiment principal. Les supports de sauvegarde sont souvent réutilisés pour réduire leur coût. Respectez toujours les directives de votre entreprise en matière de rotation des supports.

Voici quelques considérations sur les sauvegardes de données :

  • Fréquence : une sauvegarde peut nécessiter beaucoup de temps. Il est parfois plus facile d'effectuer une sauvegarde complète tous les mois ou toutes les semaines, puis de configurer des sauvegardes partielles et fréquentes des données qui ont été modifiées depuis la dernière sauvegarde complète. Toutefois, disposer d'un grand nombre de sauvegardes partielles augmente la durée nécessaire à la restauration des données.

  • Stockage : pour plus de sécurité, les sauvegardes doivent être stockées sur un site dédié et agréé. Le transfert des sauvegardes doit être effectué une fois par jour, par semaine ou par mois, conformément à la stratégie de sécurité.

  • Sécurité : les sauvegardes peuvent être protégées par des mots de passe. Le mot de passe est alors nécessaire pour restaurer les données stockées sur le support.

  • Validation : les sauvegardes doivent impérativement être vérifiées afin de garantir l'intégrité des données.

Pour effectuer une sauvegarde des données dans Windows 7 ou Windows Vista (voir l'illustration 1), sélectionnez :

Panneau de configuration > Sauvegarder et restaurer

Dans cette fenêtre, vous pouvez sauvegarder un disque dur sur un disque amovible, créer une image système ou créer un disque de réparation système.

Dans Windows 8.0 et 8.1, l'élément du Panneau de configuration Sauvegarder et restaurer a été supprimé. À la place, la fonction Historique des fichiers est utilisée pour sauvegarder les fichiers. Historique des fichiers se trouve dans le Panneau de configuration. À l'illustration 2, aucun lecteur utilisable n'a été trouvé pour la sauvegarde. Vous devez d'abord installer un lecteur Historique des fichiers, puis activer la fonction Historique des fichiers. Vous pouvez choisir un lecteur externe déjà connecté ou bien le disque interne, en cliquant sur Sélectionner un lecteur dans le volet de gauche. Vous pouvez également Sélectionner un emplacement réseau, voir l'illustration 2.
