Pare-feu logiciel

Un pare-feu logiciel est un programme qui s'exécute sur un ordinateur en vue d'autoriser ou de refuser du trafic entre un ordinateur et les autres ordinateurs auxquels il est connecté. Il applique un ensemble de règles aux transmissions de données grâce à l'inspection et au filtrage des paquets de données. Le Pare-feu Windows, voir l'illustration 1, est un pare-feu logiciel. Il est installé par défaut en même temps que le système d'exploitation.

Vous pouvez contrôler le type de données envoyées à l'ordinateur et le type de données qu'il envoie en sélectionnant les ports qui restent ouverts et ceux qui sont bloqués. Les pare-feu peuvent bloquer les connexions réseau entrantes et sortantes, sauf si des exceptions ont été définies pour ouvrir et fermer les ports requis par un programme.

Pour activer ou désactiver un port dans le Pare-feu Windows sous Windows 7, 8.0 ou 8.1, procédez comme suit :

Étape 1. Panneau de configuration > Pare-feu Windows > Paramètres avancés.

Étape 2. Choisissez de configurer les règles de trafic entrant ou les règles de trafic sortant dans le volet de gauche et cliquez sur Nouvelle règle dans le volet de droite, voir l'illustration 2.

Étape 3. Cliquez sur le bouton Port, puis cliquez sur Suivant.

Étape 4. Choisissez TCP ou UDP.

Étape 5. Sélectionnez Tous les ports locaux ou Ports locaux spécifiques pour définir certains ports individuels ou une plage de ports, puis cliquez sur Suivant.

Étape 6. Sélectionnez Bloquer la connexion, puis cliquez sur Suivant.

Étape 7. Déterminez quand cette règle doit s'appliquer, puis cliquez sur Suivant.

Étape 8. Donnez un nom à cette règle, et éventuellement une description, puis cliquez Terminer.

Pour activer ou désactiver un port dans le Pare-feu Windows sous Windows Vista, procédez comme suit :

Étape 1. Panneau de configuration > Pare-feu Windows, puis cliquez sur Autoriser un programme via le Pare-feu Windows.

Étape 2. Cliquez sur Ajouter un port et configurez le nom, le numéro de port et le protocole (TCP ou UDP).

Étape 3. Cliquez sur OK.
