Navigation InPrivate

Les navigateurs Web enregistrent des informations sur les pages que vous visitez, les recherches que vous effectuez, les noms d'utilisateur, les mots de passe et d'autres informations permettant de vous identifier. Ces fonctionnalités sont pratiques lorsque l'on utilise un ordinateur domestique sécurisé par un mot de passe. Cependant, elles peuvent poser problème lorsque vous utilisez un ordinateur public, par exemple dans une bibliothèque, dans le centre d'affaires d'un hôtel ou dans un cybercafé.

Les informations enregistrées par les navigateurs Web peuvent être récupérées et exploitées par d'autres personnes. Elles peuvent alors utiliser ces informations pour usurper votre identité, vous voler de l'argent ou modifier vos mots de passe les plus sensibles.

Pour renforcer la sécurité sur les ordinateurs publics, les navigateurs Web permettent de naviguer sur Internet de façon anonyme, sans conserver les informations. Par exemple, la fonctionnalité de navigation InPrivate d'Internet Explorer 11 empêche le transfert des informations affichées à l'illustration 1. Pendant la navigation, le navigateur InPrivate stocke temporairement les fichiers et les cookies, et les supprime lorsque la session InPrivate se termine.

Il est possible d'ouvrir une fenêtre de navigation InPrivate depuis le bureau Windows ou le navigateur.

Pour lancer la navigation InPrivate sous Windows 7, cliquez avec le bouton droit sur l'icône Internet Explorer indiquée à l'illustration 2, puis cliquez sur Démarrer la navigation InPrivate.

Une nouvelle fenêtre de navigation InPrivate s'ouvre, voir l'illustration 3. Cette fenêtre explique la fonction InPrivate et confirme son activation. Notez également que la barre d'adresses identifie désormais cette fenêtre comme une fenêtre InPrivate, comme illustré ci-contre. La fermeture de la fenêtre InPrivate met un terme à la session de navigation InPrivate. Il est important de noter que seuls cette fenêtre de navigateur et les nouveaux onglets ouverts dans cette fenêtre assurent votre confidentialité. Les autres fenêtres du navigateur ne sont pas protégées par la navigation InPrivate.

Pour ouvrir une fenêtre de navigation InPrivate dans Internet Explorer 11, procédez comme suit (voir l'illustration 4) :

Sélectionnez Outils > Navigation InPrivate.

Vous pouvez également appuyer sur Ctrl+Maj+P pour ouvrir une fenêtre InPrivate.
