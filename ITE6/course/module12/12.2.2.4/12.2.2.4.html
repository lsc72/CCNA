Filtre SmartScreen

Les navigateurs Web offrent également d'autres fonctionnalités de filtrage Web. Par exemple, Internet Explorer 11 contient la fonction de filtre SmartScreen. Cette fonction permet de détecter les sites Web d'hameçonnage, d'analyser les sites Web pour y détecter les éléments suspects et de vérifier les téléchargements à l'aide d'une liste de sites connus comme étant malveillants.

Pour activer la fonction de filtre SmartScreen dans Internet Explorer 11, sélectionnez (illustration 1) :

Outils > Filtre SmartScreen > Activer le filtre SmartScreen

Cette commande permet d'ouvrir la fenêtre Filtre SmartScreen Microsoft, voir l'illustration 2. Dans cette fenêtre, l'utilisateur peut activer ou désactiver cette fonction.

Pour l'activer, cliquez sur Activer le filtre SmartScreen (recommandé), puis cliquez sur OK.

Le contenu de la page Web peut également être vérifié. Pour analyser la page Web affichée, sélectionnez :

Sélectionnez Outils > Filtre SmartScreen > Vérifier ce site Web.

Pour signaler un site Web suspect, sélectionnez :

Sélectionnez Outils > Filtre SmartScreen > Signaler un site Web d'hameçonnage.
