BitLocker Windows

Vous pouvez également choisir de chiffrer l'intégralité d'un disque dur avec la fonction BitLocker. Pour l'utiliser, vous devez disposer d'au moins deux volumes sur un disque dur. Le volume système ne peut pas être chiffré et doit faire au moins 100 Mo. C'est celui qui contient les fichiers dont Windows a besoin pour démarrer.

Avant d'utiliser BitLocker, vous devez activer la puce TPM (Trusted Platform) dans le BIOS. Le TPM est une puce spécialisée installée sur la carte mère. Le TPM stocke des informations spécifiques au système hôte, telles que les clés de chiffrement, les certificats numériques et les mots de passe. Certaines applications comme BitLocker, qui utilisent le chiffrement, peuvent utiliser la puce TPM. Pour activer le TPM, procédez comme suit :

Étape 1. Démarrez l'ordinateur et accédez à la configuration du BIOS.

Étape 2. Recherchez l'option TPM dans les écrans de configuration du BIOS. Consultez le manuel de votre carte mère pour savoir quel écran sélectionner. L'utilitaire de configuration du ThinkPad de Lenovo est indiqué à l'illustration 1.

Étape 3. Sélectionnez Activer la puce de sécurité.

Étape 4. Enregistrez les modifications de la configuration du BIOS.

Étape 5. Redémarrez l'ordinateur.

Pour activer BitLocker dans toutes les versions de Windows, procédez comme suit :

Étape 1. Cliquez sur Panneau de configuration > Chiffrement de lecteur BitLocker.

Étape 2. Sur la page Chiffrement de lecteur BitLocker, cliquez sur Activer BitLocker pour le volume hébergeant le système d'exploitation.

Étape 3. Si le TPM n'est pas initialisé, l'Assistant d'initialisation du TPM s'affiche. Suivez les instructions pour initialiser le TPM. Redémarrez l'ordinateur.

Étape 4. La page Enregistrer le mot de passe de récupération offre les options suivantes :

  • Enregistrer le mot de passe sur un lecteur USB : le mot de passe est enregistré sur un périphérique USB.

  • Enregistrer le mot de passe dans un dossier : le mot de passe est enregistré sur un lecteur réseau ou dans un autre emplacement.

  • Imprimer le mot de passe : le mot de passe est imprimé.

Étape 5. Après avoir enregistré le mot de passe de récupération, cliquez sur Suivant.

Étape 6. Sur la page Chiffrement de lecteur BitLocker, activez la case à cocher Exécuter le test système BitLocker.

Étape 7. Cliquez sur Continuer.

Étape 8. Cliquez sur Redémarrer maintenant.

Étape 9. La barre de progression Chiffrement en cours s'affiche. Après le redémarrage de votre ordinateur, vous pouvez vérifier si BitLocker est actif (voir l'illustration 2). Vous pouvez cliquer sur Gestion TPM pour afficher les détails sur la puce TPM, voir l'illustration 3.

Remarque : le chiffrement de lecteur BitLocker peut également être utilisé avec des supports amovibles, avec BitLocker To Go. BitLocker To Go n'utilise pas de puce TPM, mais il propose un chiffrement des données et nécessite un mot de passe.
