The figure on this page is a table listing the following comparison between the MBR and GPT partitioning schemes. MBR: Maximum of 4 primary partitions, maximum partition size is 2TB, no partition table backup, partition and boot data stored in one place, any computer can boot from MBR. GPT: Maximum of 128 partitions in Windows, maximum partition size of 9.4 ZB (9.4 times 10 to the power of 21 bytes), stores partition table backup, partition and boot data stored in multiple locations across the disk, computer must be UEFI based and run 64-bit OS.
