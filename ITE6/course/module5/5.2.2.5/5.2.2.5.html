Options de récupération du système

En cas de défaillance du système, les utilisateurs peuvent utiliser les outils de récupération suivants :

  • Options de démarrage avancées de Windows (Windows 8.x)

  • Options de récupération du système (Windows 7 et Windows Vista)

  • Partition de récupération d'usine

Options de démarrage avancées de Windows 8.x

Les options de démarrage avancées sont un ensemble d'outils pour Windows 8.x permettant aux utilisateurs de dépanner, de récupérer ou de restaurer un système d'exploitation incapable de démarrer. Les options avancées font partie de l'environnement de récupération Windows (WinRE). WinRE est une plate-forme de récupération basée sur l'environnement de préinstallation Windows (WinPE). Pour accéder aux outils avancés, appuyez sur F8 durant le processus de démarrage pour l'interrompre. Dans la fenêtre Choisir une option (illustration 1), sélectionnez Dépannage. Dans la fenêtre Dépannage (illustration 2), sélectionnez Options avancées. L'illustration 3 montre la fenêtre Options avancées avec les outils suivants :

  • Restauration du système : cet outil permet de restaurer l'ordinateur à un point de restauration antérieur. Cette opération est identique à l'utilisation de l'outil Restauration du système dans Windows.

  • Récupération de l'image système : cet outil permet de restaurer l'ordinateur à l'aide d'un fichier d'image système.

  • Réparation automatique : cet outil analyse le système et tente de résoudre automatiquement les problèmes qui empêchent le démarrage correct de Windows.

  • Invite de commandes : cet outil ouvre l'invite de commandes de l'environnement de récupération, qui permet d'accéder à un certain nombre d'outils de dépannage en ligne de commande.

  • Changer les paramètres du micrologiciel UEFI : cet outil n'est visible que si l'ordinateur prend en charge l'UEFI. Utilisez cet outil pour modifier les paramètres UEFI de l'ordinateur.

  • Paramètres de démarrage de Windows : cette option permet d'activer le mode sans échec. Vous pouvez également désactiver le redémarrage automatique en cas d'échec afin de voir le message d'erreur affiché sur l'écran bleu.

Windows 7 et Windows Vista

Semblables aux options avancées de Windows 8.x, les options de récupération du système sont un ensemble d'outils de dépannage pour Windows 7 et Windows Vista. Comme le montre l'illustration 4, les options de récupération du système font également partie de l'environnement de récupération Windows.

Comme pour Windows 8.x, dans Windows 7 et Windows Vista, vous pouvez accéder à l'environnement de récupération Windows en maintenant la touche F8 enfoncée lors du démarrage de l'ordinateur. Sur l'écran Options de démarrage avancées, sélectionnez l'option Réparer votre ordinateur, puis appuyez sur Entrée pour accéder aux options de récupération du système. Vous pouvez alors utiliser les outils de récupération système pour réparer les erreurs empêchant le démarrage. Les outils suivants sont disponibles dans le menu Options de récupération système :

  • Réparation du démarrage : analyse le disque dur à la recherche d'éventuels problèmes et corrige automatiquement les erreurs liées aux fichiers système endommagés ou manquants, qui empêchent le démarrage de Windows.

  • Restauration du système : utilise les points de restauration pour restaurer les fichiers système Windows à un état antérieur.

  • Récupération de l'image système : cet outil permet de restaurer une image préalablement enregistrée sur le disque.

  • Diagnostic de mémoire Windows : cet outil examine la mémoire de l'ordinateur pour détecter les dysfonctionnements et diagnostiquer les problèmes éventuels.

  • Invite de commandes : ouvre une fenêtre d'invite de commandes où l'outil bootrec.exe peut être utilisé pour résoudre les problèmes de démarrage de Windows.

Si vous utilisez un disque de récupération, vérifiez que l'architecture utilisée est bien la même que celle du système d'exploitation à récupérer. Par exemple, si l'ordinateur exécute une version 64 bits de Windows 7, le disque de récupération doit également utiliser une architecture 64 bits.
