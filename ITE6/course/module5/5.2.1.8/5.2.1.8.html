Création des comptes

Lorsque les utilisateurs tentent de se connecter à un périphérique ou d'accéder aux ressources système, Windows utilise la procédure d'authentification pour vérifier l'identité des utilisateurs. L'authentification a lieu lorsque les utilisateurs entrent un nom d'utilisateur et un mot de passe afin de se connecter à un compte d'utilisateur. Les systèmes d'exploitation Windows utilisent l'authentification unique (SSO), qui permet aux utilisateurs de se connecter une seule fois pour accéder à toutes les fonctionnalités système, au lieu de demander aux utilisateurs de se connecter chaque fois qu'ils ont besoin d'accéder à une ressource.

Les comptes d'utilisateurs permettent à plusieurs utilisateurs de partager un seul ordinateur, tout en utilisant leurs propres fichiers et paramètres. Windows 8.1, Windows 7 et Windows Vista possèdent trois types de comptes d'utilisateurs : administrateur, standard et invité. Chaque type de compte donne à l'utilisateur un niveau de contrôle différent sur les ressources système.

Dans Windows 8.1, le compte créé durant le processus d'installation dispose de droits d'administrateur. Un utilisateur possédant des droits d'administrateur peut effectuer des modifications qui affectent tous les utilisateurs de l'ordinateur : par exemple, modifier les paramètres de sécurité ou installer des logiciels pour tous les utilisateurs. Les comptes administrateur ne doivent être utilisés que pour gérer un ordinateur et non pour une utilisation standard. En effet, ce type de compte permet d'effectuer des modifications importantes qui peuvent affecter l'ensemble des utilisateurs. De plus, les pirates cherchent à utiliser des comptes administrateur, car ils offrent plus de possibilités. Pour cette raison, il est conseillé de créer un compte d'utilisateur standard pour les tâches courantes.

Les comptes d'utilisateur standard peuvent être créés à tout moment par l'Administrateur. Un compte d'utilisateur standard peut utiliser la plupart des fonctionnalités d'un ordinateur, mais ne peut pas apporter de modifications qui affectent les autres utilisateurs ou la sécurité de l'ordinateur. Par exemple, un utilisateur standard n'a pas le droit d'installer une imprimante.

Les personnes n'ayant pas de compte d'utilisateur sur l'ordinateur peuvent utiliser un compte invité. Les comptes invité ont des autorisations limitées et doivent être activés par un administrateur. L'option du panneau de configuration Gérer les comptes sert à gérer les comptes des utilisateurs. Le Panneau de configuration est un emplacement de Windows qui contient de nombreux outils relatifs à la configuration des fenêtres. Il existe de nombreuses manières d'accéder au Panneau de configuration dans chaque version de Windows.

Pour y accéder dans Windows 7 et Windows Vista, cliquez sur le bouton Démarrer situé à gauche de la barre des tâches dans le bas du Bureau. Le menu Démarrer s'affiche alors. Cliquez sur Panneau de configuration dans le menu Démarrer. Vous pouvez également cliquer sur le bouton Démarrer et taper panneau dans la zone Rechercher les programmes et fichiers, puis appuyer sur Entrée.

L'interface de Windows 8 diffère très fortement de celles de Windows 7 et Windows Vista. Pour accéder au Panneau de configuration à partir du Bureau de Windows 8, placez le curseur dans le coin supérieur droit du Bureau et attendez que la barre d'icônes s'affiche. Cliquez sur l'icône Rechercher, tapez « panneau » dans la zone de recherche, puis appuyez sur Entrée. Sur un écran tactile, vous pouvez également faire glisser votre doigt sur l'écran à partir du côté droit pour afficher les icônes.

Par défaut, l'interface de Windows 8 ne possède pas de menu Démarrer. Windows 8 utilise un écran Démarrer à la place. Pour accéder à l'écran Démarrer, accédez tout d'abord à la barre d'icônes, puis cliquez sur l'icône Démarrer. Pour accéder au Panneau de configuration à partir de l'écran Démarrer, tapez panneau et appuyez sur Entrée. Vous pouvez également accéder à l'écran Démarrer de Windows 8.1 en cliquant sur le bouton Démarrer situé à gauche de la barre des tâches dans le bas du Bureau.

Il y a plusieurs façons d'afficher le Panneau de configuration, selon la version de Windows que vous utilisez. Tout au long de ce cours, les chemins d'accès aux outils du Panneau de configuration supposent que l'affichage est défini sur les grandes ou les petites icônes. Pour cela, choisissez Grandes icônes ou Petites icônes dans le menu déroulant Affichage par : du Panneau de configuration. Dans Windows Vista, il est possible d'afficher les icônes en cliquant sur Affichage classique dans le Panneau de configuration.

Tout au long de ce cours, les chemins d'accès aux outils du Panneau de configuration commencent par Panneau de configuration.

Pour gérer un compte d'utilisateur dans Windows 8.1 ou Windows 8, utilisez le chemin d'accès suivant :

Panneau de configuration > Comptes d'utilisateurs > Gérer les comptes

Pour créer ou supprimer un compte d'utilisateur sur Windows 7 et Windows Vista, sélectionnez :

Démarrer > Panneau de configuration > Comptes d'utilisateurs > Ajouter ou supprimer des comptes d'utilisateurs
