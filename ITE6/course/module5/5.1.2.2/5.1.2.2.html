Systèmes d'exploitation de réseau

Les systèmes d'exploitation de réseau comportent des fonctionnalités visant à optimiser l'utilisation et la gestion dans des environnements réseau. Ils présentent les caractéristiques suivantes :

  • Prise en charge de plusieurs utilisateurs

  • Exécution d'applications multiutilisateurs

  • Meilleure sécurité que les systèmes d'exploitation de bureau

Les systèmes d'exploitation de réseau offrent les ressources réseau suivantes :

  • Applications serveur, par exemple des bases de données partagées

  • Stockage de données centralisé

  • Référentiel centralisé de comptes d'utilisateurs et de ressources sur le réseau

  • File d'attente d'impression réseau

  • Systèmes de stockage redondants, comme la technologie RAID et les sauvegardes

Windows Server est un exemple de système d'exploitation de réseau.
