Migration des données

Lorsqu'une nouvelle installation est requise, les données de l'utilisateur doivent être transférées de l'ancien système d'exploitation vers le nouveau. Trois outils sont disponibles pour effectuer cette migration. L'outil à utiliser dépend de votre niveau d'expérience et de vos besoins.

Outil de migration USMT (User State Migration Tool)

L'outil de migration USMT transfère tous les fichiers et paramètres de l'utilisateur vers le nouveau système d'exploitation (voir l'illustration 1). Téléchargez et installez USMT à partir du site Web de Microsoft. Utilisez alors ce logiciel pour stocker les fichiers et paramètres de l'utilisateur en dehors du système d'exploitation. Une fois le nouveau système d'exploitation installé, téléchargez l'outil de migration USMT, puis réinstallez-le pour charger les fichiers et paramètres de l'utilisateur sur le nouveau système d'exploitation.

Remarque : l'outil USMT version 5 prend en charge la migration de données à partir de Windows 8, 7 et Vista vers Windows 8, 7 et Vista, éditions 32 ou 64 bits.

Transfert de fichiers et paramètres Windows

Si un utilisateur change d'ordinateur, vous pouvez utiliser le Transfert de fichiers et paramètres Windows pour transférer ses fichiers et paramètres. Vous pouvez effectuer cette opération à l'aide d'un câble USB, d'un CD ou d'un DVD, d'un lecteur Flash USB, d'un disque dur externe ou d'une connexion réseau.

Utilisez le Transfert de fichiers et paramètres Windows pour transférer des informations vers un ordinateur exécutant Windows 8.1 à partir d'ordinateurs utilisant l'un des systèmes d'exploitation suivants :

  • Windows 8

  • Windows 7

  • Windows Vista

Le Transfert de fichiers et paramètres Windows ne peut pas sauvegarder des fichiers sur un ordinateur exécutant Windows 8.1 pour les transférer vers un autre ordinateur. Pour sauvegarder et transférer des fichiers à partir d'un ordinateur exécutant Windows 8.1, transférez manuellement les fichiers à l'aide de périphériques de stockage externes.

Après avoir exécuté le Transfert de fichiers et paramètres Windows, vous pouvez consulter le journal répertoriant les fichiers transférés.

Pour accéder au Transfert de fichiers et paramètres Windows sur Windows 8.1 ou Windows 8, procédez comme suit :

Dans l'écran de démarrage, tapez « transfert de fichiers », puis sélectionnez Transfert de fichiers et paramètres Windows dans les résultats de la recherche.

Pour accéder au Transfert de fichiers et paramètres Windows sur Windows 7 ou Windows Vista, procédez comme suit :

Démarrer > Tous les programmes > Accessoires > Outils système > Transfert de fichiers et paramètres Windows
