Registre Windows

Les clés du Registre Windows jouent un rôle important dans le processus de démarrage de Windows. Ces clés se reconnaissent à leurs noms bien distincts, qui commencent par HKEY_, comme le montre l'illustration ci-contre. Les mots et les lettres qui suivent HKEY_ représentent la partie du système d'exploitation qui est contrôlée par cette clé. Chaque paramètre de Windows, l'arrière-plan du bureau, la couleur des boutons ou les licences octroyées pour les applications, est enregistré dans le Registre. Lorsqu'un utilisateur change les paramètres du Panneau de configuration, les associations de fichiers, les stratégies système ou les logiciels installés, les modifications sont enregistrées dans le Registre.

Chaque compte d'utilisateur a une section unique dans le Registre. Le processus d'ouverture de session de Windows charge les paramètres système du Registre pour reconfigurer le système en fonction du compte d'utilisateur sélectionné.

Le Registre est également chargé de l'enregistrement de l'emplacement des fichiers DLL (Dynamic Link Library ou bibliothèque de liens dynamiques). Un fichier DLL est constitué d'un code de programme pouvant être utilisé par différents programmes pour effectuer des fonctions courantes. Les fichiers DLL sont très importants pour le fonctionnement d'un système d'exploitation et pour les applications installées par les utilisateurs.

Pour s'assurer qu'un fichier DLL peut être localisé par le système d'exploitation ou par un programme, ce fichier doit être enregistré. Généralement, l'enregistrement est effectué automatiquement lors de la phase d'installation. Un utilisateur peut avoir besoin d'enregistrer un fichier DLL manuellement en cas de problème. L'enregistrement d'un fichier DLL relie le chemin au fichier. Cela facilite la localisation des fichiers nécessaires par les programmes.

Pour enregistrer un fichier DLL dans Windows 8.0 et 8.1 à l'aide de l'outil en ligne de commande, procédez comme suit :

Déplacez la souris vers le coin supérieur droit de l'écran > la barre d'icônes s'affiche > Rechercher, tapez commande > cliquez sur l'invite de commandes > tapez regsvr32filename.dll.

Pour enregistrer un fichier DLL dans Windows 7 et Windows Vista à l'aide de l'outil en ligne de commande, procédez comme suit :

Cliquez sur Démarrer > tapez « commande » dans la zone Rechercher les programmes et fichiers > tapez regsvr32filename.dll

Remarque : le nom du fichier DLL doit inclure le chemin d'accès complet au fichier, par exemple C:\Windows\System32\wuapi.dll.
