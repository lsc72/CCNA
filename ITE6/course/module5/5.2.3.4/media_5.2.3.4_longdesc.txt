The figure on this page is a table listing the following Windows Registry Keys. HKEY: HKEY_CLASSES_ROOT, Description: Information about which file extensions map to a particular application. HKEY: HKEY_CURRENT_USER, Description: Information, such as desktop settings and history, related to the current user of a PC. HKEY: HKEY_USERS, Description: Information about all users who have logged onto a system. HKEY: HKEY_LOCAL_MACHINE, Description: Information relating to the hardware and software. HKEY: HKEY_CURRENT_CONFIG, Description: Information relating to all active devices on a system.
