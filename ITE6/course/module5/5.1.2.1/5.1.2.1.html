Systèmes d'exploitation de bureau

Un technicien peut avoir à choisir et à installer un système d'exploitation pour un client. Il existe deux types de systèmes d'exploitation, à savoir les systèmes d'exploitation de bureau et les systèmes d'exploitation de réseau. Un système d'exploitation de bureau est conçu pour une utilisation domestique ou dans de petites entreprises, avec un nombre limité d'utilisateurs. Un système d'exploitation de réseau est utilisé dans un environnement de grande entreprise, avec de nombreux utilisateurs et de multiples besoins.

Un système d'exploitation de bureau présente les caractéristiques suivantes :

  • Il prend en charge un seul utilisateur.

  • Il exécute des applications mono-utilisateur.

  • Il permet de partager des fichiers et des dossiers sur de petits réseaux à la sécurité limitée.

Sur le marché actuel des logiciels, il existe trois grandes catégories de systèmes d'exploitation de bureau : Microsoft Windows, Apple Mac OS et Linux. Ce chapitre se concentre sur les systèmes d'exploitation Microsoft :

  • Windows 8.1 : Windows 8.1 est une mise à jour de Windows 8. Cette mise à jour inclut des améliorations destinées à faciliter l'utilisation de Windows pour les personnes qui utilisent des interfaces tactiles aussi bien qu'une souris et un clavier.

  • Windows 8 : Windows 8 a introduit l'interface utilisateur Metro qui unifie l'apparence de Windows sur les ordinateurs de bureau, les ordinateurs portables, les téléphones mobiles et les tablettes. Cette interface permet aux utilisateurs d'interagir avec le système d'exploitation au moyen d'un écran tactile ou d'un clavier et d'une souris. Une autre version, Windows 8 Pro, est destinée aux professionnels des entreprises et aux techniciens, et offre des fonctionnalités supplémentaires.

  • Windows 7 : Windows 7 est une mise à niveau de Windows XP ou de Windows Vista. Cette version est conçue pour être exécutée sur des ordinateurs personnels. Elle offre une interface utilisateur graphique améliorée et de meilleures performances que les versions précédentes.

  • Windows Vista : Windows Vista est un système d'exploitation pour ordinateur personnel. Successeur de Windows XP, il offre une sécurité améliorée et a introduit l'interface utilisateur Windows Aero.
