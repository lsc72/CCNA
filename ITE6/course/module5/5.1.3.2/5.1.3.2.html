Configuration matérielle minimale et compatibilité avec les plates-formes de système d'exploitation

Pour pouvoir être installés et fonctionner convenablement, les systèmes d'exploitation exigent une configuration matérielle minimale.

Identifiez l'équipement de votre client. Si des mises à niveau matérielles sont nécessaires pour atteindre la configuration minimale requise et pouvoir installer un système d'exploitation, procédez à une analyse des coûts pour déterminer la meilleure action à entreprendre. Dans certains cas, il peut être moins coûteux pour le client d'acheter un nouvel ordinateur que de mettre à niveau son système. Dans d'autres cas, il peut s'avérer plus économique de mettre à niveau un ou plusieurs des composants suivants :

  • Mémoire vive (RAM)

  • Disque dur

  • CPU

  • Carte vidéo

  • Carte mère

Remarque : si une application requiert une configuration matérielle plus puissante que le système d'exploitation, vous devez respecter la configuration minimale requise par l'application pour que cette dernière fonctionne correctement.

Une fois que vous avez défini la configuration matérielle minimale nécessaire à un système d'exploitation, assurez-vous que tous les composants de l'ordinateur sont compatibles avec le système d'exploitation que vous avez sélectionné pour votre client.

Centre de compatibilité Microsoft

Microsoft gère un Centre de compatibilité en ligne, qui permet aux techniciens de vérifier la compatibilité des logiciels et du matériel avec les systèmes d'exploitation, comme indiqué sur l'illustration ci-contre. Cet outil répertorie tout le matériel testé et certifié compatible avec Windows. Si certains composants du client ne figurent pas sur cette liste, cela signifie qu'ils ont sans doute besoin d'une mise à niveau.

Cliquez ici pour accéder au Centre de compatibilité Microsoft.

Pour vérifier la compatibilité de Windows sur la page des résultats, cliquez sur votre produit afin d'afficher la page des détails des produits. Le menu déroulant situé à côté du champ État de compatibilité de vous permet de sélectionner la version de Windows qui vous intéresse et de vérifier la compatibilité.
