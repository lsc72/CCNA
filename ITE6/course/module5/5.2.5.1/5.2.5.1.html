Structures des répertoires

Sous Windows, les fichiers sont organisés en structure de répertoires. Une structure de répertoires est conçue pour stocker les fichiers système, les fichiers des utilisateurs et les fichiers des programmes. Le niveau racine de la structure de répertoires Windows, à savoir la partition, est généralement appelé « C » (illustration 1). Le lecteur C contient un ensemble de répertoires standard, appelés dossiers, pour le système d'exploitation, les applications, les informations de configuration et les fichiers de données. Les répertoires peuvent contenir des sous-répertoires, appelés des sous-dossiers.

Si Windows ne reconnaît pas les systèmes de fichiers présents sur un nouveau disque, ce dernier doit être initialisé et formaté avant de pouvoir être utilisé. Lors de l'initialisation du disque, les informations relatives à l'emplacement des fichiers sur le disque sont supprimées. Sans ces informations, le disque apparaît comme étant vierge pour le système d'exploitation. Durant le formatage, les données présentes sur le disque sont supprimées et ne sont par conséquent plus accessibles. En outre, les secteurs défectueux du disque sont remappés et une structure de système de fichiers est créée pour le système d'exploitation lors du formatage.

Après l'installation initiale, les utilisateurs peuvent installer la plupart des applications et données dans le répertoire de leur choix. Le programme d'installation de Windows crée des répertoires ayant des objectifs spécifiques, comme le stockage de photos ou de fichiers de musique. Lorsque les fichiers d'un même type sont enregistrés dans un emplacement spécifique, l'utilisateur peut plus facilement les retrouver.

Remarque : même si l'initialisation d'un disque détruit normalement toutes les données qui y étaient stockées, certaines entreprises sont spécialisées dans la récupération de données à partir de disques ayant été réinitialisés.

Remarque : nous vous recommandons de stocker les fichiers dans des dossiers et des sous-dossiers plutôt qu'au niveau racine d'un disque.

Montage d'un volume

Vous pouvez mapper un lecteur à un dossier vide sur un volume. Cette opération s'appelle le montage d'un lecteur. Les lecteurs montés sont affectés à des chemins de lecteurs au lieu de lettres, et ils sont affichés sous la forme d'une icône de lecteur dans l'Explorateur Windows. L'Explorateur Windows est un outil permettant aux utilisateurs de voir tous les lecteurs, dossiers et fichiers d'un ordinateur de façon structurée. Utilisez un lecteur monté pour configurer plus de 26 lecteurs dans l'ordinateur, ou lorsque vous avez besoin d'espace de stockage supplémentaire sur un volume.

Pour monter un volume sous Windows 8.x, procédez comme suit :

Étape 1. Ouvrez Gestion des disques et cliquez avec le bouton droit n'importe où sur un emplacement vide.

Étape 2. Sélectionnez Nouveau volume simple pour démarrer l'Assistant Création d'un volume simple (illustration 2).

Étape 3. Spécifiez la taille de la nouvelle partition.

Étape 4. Cliquez sur Suivant.

Étape 5. Sélectionnez Monter dans le dossier NTFS vide suivant : et spécifiez l'emplacement du dossier vide.

Étape 6. Choisissez si le volume doit être formaté et indiquez quel système de fichiers utiliser.

Étape 7. Finalisez et fermez l'outil Gestion des disques.

Pour monter un volume sous Windows 7 ou Windows Vista, procédez comme suit :

Étape 1. Sélectionnez Démarrer > Panneau de configuration > Outils d'administration > Gestion de l'ordinateur.

Étape 2.Cliquez sur Gestion des disques dans le volet gauche.

Étape 3. Cliquez avec le bouton droit sur la partition ou le volume à monter.

Étape 4. Cliquez sur Modifier la lettre de lecteur et les chemins d'accès.

Étape 5. Cliquez sur Ajouter.

Étape 6. Cliquez sur Monter dans le dossier NTFS vide suivant.

Étape 7. Accédez à un dossier vide sur un volume NTFS ou créez-en un, puis cliquez sur OK.

Étape 8. Fermez la fenêtre Gestion de l'ordinateur.
