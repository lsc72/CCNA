Emplacement des fichiers système et des fichiers de l'utilisateur

Emplacement des fichiers de l'utilisateur

Par défaut, Windows stocke la plupart des fichiers créés par les utilisateurs dans le dossier C:\Utilisateurs\Nom_utilisateur\. Le dossier de chaque utilisateur contient des sous-dossiers pour la musique, les vidéos, les sites Web et les images, entre autres (voir l'illustration 1). De nombreux programmes stockent également des données spécifiques des utilisateurs dans ce dossier. Si plusieurs utilisateurs partagent un même ordinateur, chaque utilisateur dispose de ses propres dossiers, qui contiennent ses favoris, les éléments du Bureau ainsi que des cookies. Les cookies sont des fichiers qui contiennent des informations provenant des sites Web visités par les utilisateurs.

Dossier système

Lorsque le système d'exploitation Windows est installé, la plupart des fichiers utilisés pour faire fonctionner l'ordinateur sont placés dans le dossier C:\Windows\system32. Le contenu du dossier système Windows est présenté à l'illustration 2.

Polices

Le dossier C:\Windows\Fonts contient les polices installées sur l'ordinateur. Les polices ont des formats différents, notamment TrueType, OpenType, Composite et PostScript. Arial, Times New Roman et Courier sont des exemples de polices de caractères. Vous pouvez accéder au dossier Polices depuis le Panneau de configuration. Vous pouvez installer des polices en double-cliquant sur le fichier des polices et en sélectionnant Installer.

Fichiers temporaires

Le dossier Fichiers temporaires contient les fichiers créés par le système d'exploitation et les programmes dont il a besoin pendant une courte durée. Par exemple, une application en cours d'installation peut créer des fichiers temporaires afin d'augmenter la mémoire vive disponible pour les autres applications.

La plupart des programmes utilisent des fichiers temporaires qui sont en général supprimés automatiquement lorsque l'application ou le système d'exploitation n'en a plus besoin. Toutefois, certains fichiers temporaires doivent être supprimés manuellement. Étant donné qu'ils utilisent de l'espace (qui pourrait être utilisé pour d'autres fichiers) sur le disque dur, il est recommandé de vérifier et, le cas échéant, de supprimer ces fichiers tous les deux ou trois mois.

Sous Windows, les fichiers temporaires sont généralement placés dans les dossiers suivants :

  • C:\Windows\Temp

  • C:\Users\User_Name\AppData\Local\Temp

  • %USERPROFILE%\AppData\Local\Temp

Remarque : %USERPROFILE% est une variable d'environnement définie par le système d'exploitation, en fonction du nom de l'utilisateur actuellement connecté à l'ordinateur. Les variables d'environnement sont utilisées par le système d'exploitation, les applications et les programmes d'installation des logiciels.

Pour afficher les variables d'environnement qui sont configurées dans Windows, sélectionnez :

Panneau de configuration > Système > Paramètres système avancés > onglet Avancé > Variables d'environnement

Program Files

Le dossier Program Files est utilisé par la plupart des programmes d'installation des applications pour installer des logiciels. Sur les systèmes 32 bits, tous les programmes sont en 32 bits et sont installés dans le dossier C:\Program Files. Sur les systèmes 64 bits, les programmes 64 bits sont installés dans le dossier C:\Program Files, tandis que les programmes 32 bits sont généralement installés dans le dossier C:\Program Files (x86).
