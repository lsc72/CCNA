Procédures de démarrage multiple

Vous pouvez faire cohabiter plusieurs systèmes d'exploitation sur un seul ordinateur. Certaines applications exigent la version la plus récente d'un système d'exploitation, alors que d'autres ont besoin d'une version antérieure. Le double démarrage permet de disposer de plusieurs systèmes d'exploitation sur un ordinateur. Lors du processus de démarrage, si le Gestionnaire de démarrage Windows (BOOTMGR) détecte plusieurs systèmes d'exploitation, l'utilisateur est invité à choisir le système d'exploitation à charger. L'illustration 1 montre le BOOTMGR de Windows 8.x. L'illustration 2 montre le BOOTMGR de Windows 7 et Windows Vista.

Pour que vous puissiez créer un système à double démarrage dans Microsoft Windows, le disque dur doit contenir plusieurs partitions.

Le système d'exploitation le plus ancien doit être installé sur la partition principale ou sur le disque dur marqué comme partition active. Le deuxième système d'exploitation doit être installé sur la deuxième partition ou le deuxième disque dur. Les fichiers de démarrage sont installés automatiquement dans la partition active.

Fichier BOOTMGR

Pendant l'installation, le fichier BOOTMGR est créé sur la partition active, afin de permettre la sélection du système d'exploitation à démarrer. Vous pouvez modifier le fichier BOOTMGR afin d'inverser l'ordre des systèmes d'exploitation. Le délai dont dispose l'utilisateur pour choisir le système d'exploitation au démarrage peut également être modifié. La durée par défaut est de 30 secondes. Ce délai s'ajoute au délai de démarrage de l'ordinateur, sauf si l'utilisateur intervient pour sélectionner lui-même un système d'exploitation. Si un seul système d'exploitation est présent sur le disque, paramétrez ce délai sur 5 ou 10 secondes pour accélérer le démarrage de l'ordinateur.

Pour modifier le délai d'affichage de la liste des systèmes d'exploitation, procédez comme suit :

Sélectionnez Démarrer > Panneau de configuration > Système et sécurité > Système > Paramètres système avancés > cliquez sur l'onglet Avancé > dans la zone Démarrage et récupération, sélectionnez Paramètres.
