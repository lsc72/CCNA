Mises à niveau des systèmes d'exploitation Windows

La mise à niveau du système d'exploitation peut être plus rapide qu'une nouvelle installation. Cependant, la procédure diffère selon les versions. Par exemple, l'utilitaire d'installation de Windows 8.1 remplace les fichiers Windows existants par des fichiers Windows 8.1. Toutefois, les applications et paramètres existants sont conservés.

La version d'un système d'exploitation détermine les options de mise à niveau disponibles. Par exemple, un système d'exploitation 32 bits ne peut pas être mis à niveau vers un système d'exploitation 64 bits. De même, Windows XP ne peut pas être mis à niveau vers Windows 8.1. Avant de tenter une mise à niveau, rendez-vous sur le site Web du développeur du système d'exploitation pour consulter la liste des mises à niveau possibles.

Remarque : avant d'effectuer une mise à niveau, sauvegardez toutes les données de l'ordinateur, au cas où il y aurait un problème pendant l'installation.

Comme le montre l'illustration ci-contre, les utilitaires de mise à niveau de Windows diffèrent très peu d'une version du système d'exploitation à l'autre. Les étapes ci-dessous sont un exemple de mise à niveau vers Windows 8.1, mais elles seraient également valables pour n'importe quelle version de Windows :

Étape 1. Insérez le disque de Windows 8.1 dans le lecteur optique. La fenêtre d'installation s'affiche.

Étape 2. Sélectionnez l'option Installer maintenant.

Étape 3. Vous êtes invité à télécharger les mises à jour importantes pour l'installation.

Étape 4. Acceptez le Contrat de Licence Utilisateur Final (CLUF), puis cliquez sur Suivant.

Étape 5. Cliquez sur Mettre à niveau. Le système commence à copier les fichiers d'installation.

Étape 6. Suivez les instructions qui s'affichent pour effectuer la mise à niveau. Une fois l'installation terminée, l'ordinateur redémarre.
