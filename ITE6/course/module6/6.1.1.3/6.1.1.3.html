Menu Démarrer

Menu Démarrer sous Windows 8.1 et Windows 8.0

Sous Windows 8.0, avec l'introduction de l'environnement des applications Windows, Microsoft a choisi de supprimer le bouton Démarrer et le menu Démarrer (voir l'illustration 1). Le menu Démarrer a été remplacé par l'écran d'accueil, qui est l'écran principal où s'affichent les vignettes.

Compte tenu des nombreuses demandes, Microsoft a remis le bouton Démarrer sous Windows 8.1 (voir l'illustration 2). L'écran d'accueil joue toujours le rôle du menu Démarrer, mais les utilisateurs de Windows 8.1 disposent à présent d'un bouton pour accéder au menu Démarrer. Pour accéder à l'écran d'accueil, vous pouvez également appuyer sur la touche Windows de votre clavier ou cliquer sur le bouton Démarrer de la barre des icônes.

Pour afficher un menu Démarrer réduit sous Windows 8.1, cliquez avec le bouton droit sur le bouton Démarrer (voir l'illustration 3).

Le menu Démarrer et la barre des tâches permettent aux utilisateurs de gérer les programmes, d'effectuer des recherches et de gérer les applications en cours d'exécution. Pour personnaliser le menu Démarrer et la barre des tâches, cliquez sur l'un de ces deux éléments et sélectionnez Propriétés.

Menu Démarrer sous Windows 7 et Windows Vista

Pour accéder au menu Démarrer, cliquez sur l'icône Windows en bas à gauche du Bureau. Le menu Démarrer (illustration 4) affiche toutes les applications installées sur l'ordinateur, la liste des documents récemment ouverts, ainsi qu'une liste d'autres éléments, par exemple, la fonction de recherche, l'aide et le panneau de configuration. Pour personnaliser le menu Démarrer sous Windows 7 ou Windows Vista :

Cliquez avec le bouton droit sur une zone vide de la barre des tâches et sélectionnez Propriétés > Menu Démarrer > Personnaliser.

Barre des tâches

Sous Windows 8 et Windows 7, les fonctionnalités suivantes ont été ajoutées à la barre des tâches, pour faciliter la navigation, l'organisation et l'accès aux fenêtres et aux notifications :

  • Liste de raccourcis - pour afficher une liste des tâches propres à une application, cliquez avec le bouton droit sur l'icône de l'application dans la barre des tâches.

  • Épinglage - pour ajouter une application à la barre des tâches, afin d'y accéder plus facilement, cliquez avec le bouton droit sur l'icône de l'application et sélectionnez Épingler à la barre des tâches.

  • Miniatures - pour afficher l'image miniature d'un programme en cours d'exécution, placez votre curseur sur l'icône du programme dans la barre des tâches.
