Installation et désinstallation des applications

En tant que technicien, vous devrez ajouter et supprimer des logiciels sur les ordinateurs de vos clients. La plupart des applications utilisent un processus d'installation automatique lorsque le disque est inséré dans le lecteur optique. L'utilisateur doit cliquer sur certaines options dans l'assistant d'installation et fournir des informations lorsqu'il y est invité.

Installation d'une application

Insérez le CD ou le DVD dans le lecteur optique ou ouvrez le fichier du programme téléchargé. Le programme d'installation démarre. S'il ne démarre pas, exécutez le fichier d'installation sur le disque pour commencer l'installation, ou téléchargez à nouveau le programme.

Une fois l'application installée, vous pouvez l'exécuter à partir du menu Démarrer ou d'une icône de raccourci installée par l'application sur le Bureau. Vérifiez que l'application fonctionne correctement. En cas de problème, réparez ou désinstallez l'application. Certaines applications, comme Microsoft Office, proposent une option de réparation dans le programme d'installation. Outre le processus décrit ci-dessus, Windows 8 permet d'accéder à Windows Store (voir l'illustration 1). Windows Store offre aux utilisateurs la possibilité de rechercher et d'installer des applications sur un ordinateur Windows 8 (ou tout autre périphérique Windows 8). Pour ouvrir l'application Windows Store :

Sur l'écran d'accueil, tapez Store et cliquez sur l'icône Store qui s'affiche.

Désinstallation ou modification d'un programme

Si une application n'est pas désinstallée correctement, vous risquez de laisser des fichiers sur le disque dur et des paramètres inutiles dans le Registre. Cela peut avoir un impact négatif sur les ressources système et sur l'espace libre du disque dur. Les fichiers inutiles peuvent également réduire la vitesse de lecture du Registre. Microsoft conseille d'utiliser l'utilitaire Programmes et fonctionnalités à chaque désinstallation, modification ou réparation d'applications. Cet utilitaire vous guide au cours du processus de suppression du logiciel et supprime tous les fichiers installés.

L'illustration 2 affiche les Programmes et fonctionnalités sous Windows 8.

L'illustration 3 affiche les Programmes et fonctionnalités sous Windows 7 et Windows Vista.

Dans certains cas, vous pouvez installer ou désinstaller des fonctions optionnelles d'un programme à l'aide de l'utilitaire Programmes et fonctionnalités. Cette option n'est pas proposée par tous les programmes.

Pour ouvrir l'utilitaire Programmes et fonctionnalités, sélectionnez :

Panneau de configuration > Programmes et fonctionnalités

Pour désinstaller ou modifier un programme :

Étape 1. Cliquez sur Panneau de configuration > Programmes et fonctionnalités.

Étape 2. Sélectionnez un programme, puis cliquez sur Désinstaller. Pour modifier un programme, cliquez sur Modifier ou Réparer. Des droits d'administrateur peuvent être nécessaires. Si vous êtes invité à entrer un mot de passe administrateur ou une confirmation, tapez le mot de passe ou entrez la confirmation.
