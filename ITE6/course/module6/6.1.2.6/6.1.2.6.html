Paramètres d'affichage

Vous pouvez également personnaliser l'apparence du Bureau en ajustant la résolution et la qualité des couleurs dans l'utilitaire Paramètres d'affichage, voir l'illustration ci-contre. Si la résolution de l'écran n'est pas correctement définie, des problèmes d'affichage peuvent être observés (en fonction des réglages des cartes graphiques et des écrans). Il est également possible de modifier d'autres paramètres d'affichage, tels que l'image d'arrière plan du bureau, l'écran de veille et d'autres options.

Si vous utilisez un écran LCD, la résolution doit être définie sur la résolution native. La résolution native définit la sortie vidéo sur le même nombre de pixels que l'écran. Si vous n'utilisez pas la résolution native, l'écran ne produit pas la meilleure image possible.

Pour modifier manuellement les paramètres d'affichage sous Windows 8 et Windows 7, sélectionnez :

Panneau de configuration > Affichage > Modifier les paramètres d'affichage

Sous Windows Vista, sélectionnez :

Panneau de configuration > Personnalisation > Paramètres d'affichage

Sous Windows 7 et Windows 8, vous pouvez ajuster les paramètres suivants :

  • Résolution d'écran - Détermine le nombre de pixels. Plus le nombre de pixels est élevé, plus la résolution est bonne.

  • Orientation - Détermine l'orientation de l'écran, à savoir Paysage, Portrait, Paysage (renversé) ou Portrait (renversé).

  • Fréquence de rafraîchissement - Détermine la fréquence à laquelle l'image est réaffichée sur l'écran. La fréquence de rafraîchissement est exprimée en hertz (Hz). Plus elle est élevée, plus l'image est stable.

  • Couleurs - Détermine le nombre de couleurs simultanément visibles à l'écran. Plus le nombre de bits est élevé, plus le nombre de couleurs est élevé. La palette 8 bits comprend 256 couleurs. La palette 16 bits comprend 65 536 couleurs. La palette 24 bits (couleurs vraies) comprend 16 millions de couleurs. La palette 32 bits contient les couleurs de la palette 24 bits, ainsi que 8 bits de données supplémentaires, par exemple pour la transparence.
