Propriétés du Bureau

De nombreux aspects de l'environnement des applications Windows 8 peuvent être personnalisés.

  • Pour réorganiser les vignettes, faites-les glisser.

  • Pour renommer un groupe de vignettes, cliquez avec le bouton droit sur une zone vide de l'écran, puis sélectionnez Nommer les groupes.

  • Pour ajouter des vignettes à l'écran principal, cliquez avec le bouton droit sur l'application Windows de votre choix après l'avoir recherchée, puis sélectionnez Épingler à l'écran d'accueil.

  • Pour rechercher une application, cliquez sur Rechercher dans la barre des icônes. Vous pouvez également taper les premières lettres du nom de l'application dans l'environnement des applications Windows. La recherche démarre automatiquement.

L'illustration 1 présente l'environnement des applications Windows 8 et la barre des icônes.

Pour personnaliser le Bureau, cliquez avec le bouton droit n'importe où sur ce dernier et choisissez Personnaliser.

Remarque : sous Windows 8, cliquez sur la vignette Bureau pour sortir de l'environnement des applications et afficher le Bureau.

Dans la fenêtre Personnalisation, vous pouvez modifier l'apparence du Bureau, ainsi que les paramètres d'affichage et les paramètres audio.
