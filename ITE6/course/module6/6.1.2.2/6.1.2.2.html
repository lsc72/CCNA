Comptes d'utilisateurs

Un compte d'administrateur est créé lors de l'installation de Windows. Pour créer un compte d'utilisateur par la suite, ouvrez l'utilitaire Comptes d'utilisateurs, voir l'illustration 1, en sélectionnant :

Panneau de configuration > Comptes d'utilisateurs.

L'utilitaire Comptes d'utilisateurs fournit des options permettant de gérer les mots de passe, de modifier les images, les noms et les types de compte, ainsi que les paramètres de contrôle du compte utilisateur.

Remarque : certaines fonctions de l'utilitaire Comptes d'utilisateurs requièrent des droits d'administration et peuvent être inaccessibles à partir d'un compte d'utilisateur standard.

Paramètres de contrôle de compte d'utilisateur

Le contrôle de compte d'utilisateur surveille les programmes de l'ordinateur et avertit les utilisateurs lorsqu'une action est susceptible de menacer l'ordinateur. Sous Windows 8 ou Windows 7, vous pouvez ajuster le niveau de surveillance du contrôle de compte d'utilisateur, voir l'illustration 2. Lorsque Windows 7 ou Windows 8 est installé, le contrôle de compte d'utilisateur du compte principal est défini par défaut sur M'avertir uniquement quand des programmes tentent d'apporter des modifications à mon ordinateur, voir l'illustration 2. Vous êtes averti quand vous effectuez des modifications à ces paramètres.

Pour régler le niveau de surveillance du contrôle de compte d'utilisateur, sélectionnez :

Panneau de configuration > Comptes d'utilisateurs > Modifier les paramètres de contrôle de compte d'utilisateur
