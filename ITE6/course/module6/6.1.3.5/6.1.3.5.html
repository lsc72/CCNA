Analyseur de performances et Diagnostic de mémoire Windows

L'Analyseur de performances (illustration 1) affiche l'analyse des performances et le résumé système. Vous devez disposer de privilèges d'administration pour accéder à la console Analyseur de performances.

Le Résumé système affiche des informations en temps réel sur les processeurs, les disques, la mémoire et l'utilisation du réseau. Utilisez le Résumé système pour afficher des données détaillées sur les ressources que vous utilisez lors de l'exécution d'une ou plusieurs tâches spécifiques. Les données affichées peuvent vous aider à comprendre l'impact de la charge de travail sur les ressources système, notamment le processeur, la mémoire et le réseau. Vous pouvez visualiser les données d'utilisation dans des histogrammes, des graphiques et des rapports. Ces données peuvent également vous aider à déterminer si une mise à niveau est nécessaire.

Pour ouvrir l'Analyseur de performances sous Windows 8 et Windows 7, sélectionnez :

Panneau de configuration > Outils d'administration > Analyseur de performances

Sous Windows Vista, sélectionnez :

Panneau de configuration > Outils d'administration > Moniteur de fiabilité et de performances

L'utilitaire Diagnostic de mémoire Windows (illustration 2) est un outil d'administration qui surveille la mémoire physique installée dans l'ordinateur pour détecter les erreurs éventuelles.

Pour accéder à l'utilitaire Diagnostic de mémoire Windows sous Windows 8 et Windows 7, procédez comme suit :

Panneau de configuration > Outils d'administration > Diagnostic de mémoire Windows

Pour accéder à l'utilitaire Diagnostic de mémoire Windows sous Windows Vista, procédez comme suit :

Panneau de configuration > Outils d'administration > Outil Diagnostics de la mémoire
