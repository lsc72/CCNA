Gestionnaire de périphériques

Le Gestionnaire de périphériques (illustration 1) affiche tous les périphériques installés sur l'ordinateur pour vous permettre de diagnostiquer et de résoudre leurs problèmes. Il permet également d'accéder à des informations détaillées sur les pilotes et le matériel installés. Les fonctions suivantes sont également disponibles :

  • Mettre à jour le pilote - modifier le pilote actuellement installé.

  • Version précédente - modifier le pilote actuellement installé pour installer la version précédente.

  • Désinstaller - supprimer le pilote.

  • Désactiver - désactiver le périphérique.

Pour accéder au Gestionnaire de périphériques, sélectionnez :

Panneau de configuration > Gestionnaire de périphériques

À partir du Gestionnaire de périphériques, double-cliquez sur le nom d'un périphérique afin d'afficher ses propriétés.

Le Gestionnaire de périphériques utilise des icônes pour indiquer les éventuels problèmes relatifs aux périphériques (voir l'illustration 2).

Périphériques et imprimantes

Périphériques et imprimantes permet de connaître précisément les périphériques connectés à un ordinateur. Les périphériques s'affichant dans Périphériques et imprimantes sont généralement des périphériques externes à connecter à votre ordinateur via un port ou une connexion au réseau. Périphériques et imprimantes permet également d'ajouter rapidement un nouveau périphérique à l'ordinateur. Windows installe automatiquement les pilotes requis.

Les périphériques répertoriés sont les suivants :

  • Les périphériques portables qui vous suivent partout et que vous connectez occasionnellement à votre ordinateur, comme les téléphones mobiles, les lecteurs de musique et les appareils photo numériques.

  • Les périphériques que vous branchez occasionnellement à un port USB de votre ordinateur, comme les disques durs USB externes, les lecteurs Flash, les webcams, les claviers et les souris.

  • Toutes les imprimantes connectées à votre ordinateur

  • Les appareils sans fil connectés à votre ordinateur, comme les périphériques Bluetooth et les périphériques USB sans fil.

  • Les périphériques réseau compatibles connectés à votre ordinateur, comme les scanners réseau, les extendeurs de supports ou les périphériques de stockage en réseau (NAS).

Pour accéder à l'utilitaire Périphériques et imprimantes sous Windows 8 et Windows 7, sélectionnez :

Panneau de configuration > Périphériques et imprimantes

Pour installer un nouveau périphérique, cliquez sur le bouton Ajouter un périphérique.

L'illustration 3 montre l'utilitaire Périphériques et imprimantes sous Windows 8.

Sous Windows Vista, la fonction Ajout de matériel permet d'ajouter de nouveaux périphériques sur votre ordinateur. Pour accéder à Ajout de matériel, sélectionnez :

Panneau de configuration > Ajout de matériel

Son

L'utilitaire Son permet de configurer les périphériques audio ou de modifier les sons de l'ordinateur. Vous pouvez par exemple remplacer le « bip » de notification de vos e-mails par un carillon. L'utilitaire Son permet également de choisir le périphérique audio à utiliser pour lire les enregistrements.

Pour accéder à l'utilitaire Son, sélectionnez :

Panneau de configuration > Son

L'illustration 4 présente l'utilitaire Son sous Windows 8.
