The figure on this page lists step 5 in troubleshooting operating system problems, verify full system functionality and implement preventive measures. Verify full functionality: shut down the computer and restart it, check Event logs to make sure there are no new warnings or errors, check Device Manager to see that there are no warnings or errors, run dxdiag to make sure Directx running correctly, make sure applications run properly, make sure networks shares are accessible, make sure the internet can be accessed, re-run system file checker to ensure all files are correct, check Task Manager to ensure that the status of all programs is running, re-run any third party diagnostic tools.
