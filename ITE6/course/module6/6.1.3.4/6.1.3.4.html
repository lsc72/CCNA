Configuration du système

Configuration système (MSCONFIG) est un outil utilisé pour identifier les problèmes empêchant Windows de démarrer correctement. Pour contribuer à isoler le problème, les services et programmes de démarrage peuvent être désactivés, puis réactivés les uns après les autres. Une fois que vous avez déterminé la cause, supprimez de manière permanente ou désactivez le programme ou le service. Vous pouvez aussi le réinstaller.

L'illustration ci-contre explique brièvement les onglets et options disponibles dans la Configuration système.

Procédez comme suit pour ouvrir la Configuration système :

Panneau de configuration > Outils d'administration > Configuration système
