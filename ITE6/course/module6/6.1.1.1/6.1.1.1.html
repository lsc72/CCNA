Bureau Windows

Une fois le système d'exploitation installé, vous pouvez personnaliser le Bureau de l'ordinateur selon vos besoins. Le Bureau d'un ordinateur est la représentation graphique de l'espace de travail. Il est communément appelé « interface graphique utilisateur » (GUI). Le bureau comporte des icônes, des barres d'outils et des menus permettant de manipuler les fichiers. Les images, les sons et les couleurs utilisés peuvent être modifiés pour personnaliser l'apparence du Bureau. L'ensemble de ces éléments personnalisables constitue un thème. L'illustration 1 présente un environnement de bureau Windows 8.

Windows 8 est doté d'un nouveau Bureau constitué de vignettes qui s'affichent sur l'écran d'accueil, comme illustré sur l'illustration 2. Cet environnement, utilisé par les ordinateurs de bureau et les ordinateurs portables, est optimisé pour les terminaux mobiles. L'écran d'accueil affiche un groupe de vignettes personnalisable, qui vous permet d'accéder aux applications et à d'autres informations, comme les actualités sur les médias sociaux et les notifications de calendrier. Ces vignettes matérialisent les notifications, applications et programmes du Bureau. Celles pouvant afficher du contenu dynamique s'appellent des vignettes dynamiques. La barre verticale, ou barre des icônes, est également un nouvel élément de l'interface graphique utilisateur. Pour accéder aux icônes, faites-les glisser avec votre doigt depuis le bord droit de l'écran ou placez le curseur de votre souris dans le coin supérieur droit de l'écran.

Windows 8 nécessite la configuration matérielle suivante :

  • Processeur 1 GHz 32 bits ou 64 bits

  • 1 Go (32 bits) ou 2 Go (64 bits) de mémoire vive

  • Carte graphique DirectX 9 prise en charge

  • Au moins 16 Go (32 bits) ou 20 Go (64 bits) d'espace disponible sur le disque dur

Le thème par défaut de Windows 7 et Windows Vista est le thème Aero. Il possède des bords de fenêtres translucides, de nombreuses animations ainsi que des icônes qui sont des images miniatures du contenu des fichiers. Ce thème requérant des capacités graphiques avancées, il n'est disponible que sur les ordinateurs ayant au minimum la configuration matérielle suivante :

  • Processeur 1 GHz 32 bits ou 64 bits

  • 1 Go de mémoire vive

  • Carte graphique 128 Mo

  • Carte graphique compatible DirectX 9 avec pilote WDDM, Pixel Shader 2.0 au niveau matériel et format 32 bits par pixel

Windows 8.1, 8.0 et 7 incluent les fonctionnalités de bureau suivantes :

  • Shake - Réduisez toutes les fenêtres non utilisées en cliquant sur la barre de titre d'une fenêtre et en « secouant » la souris tout en maintenant le bouton droit enfoncé. Répétez l'opération pour agrandir toutes les fenêtres.

  • Peek - Affichez les icônes du Bureau qui sont à l'arrière des fenêtres en plaçant votre curseur sur le bouton Afficher le Bureau situés sur le bord droit de la barre des tâches. Les fenêtres ouvertes deviennent transparentes. Cliquez sur ce bouton pour réduire toutes les fenêtres.

  • Snap - Redimensionnez une fenêtre en la faisant glisser vers un bord de l'écran. Déplacez la fenêtre vers le bord gauche de l'écran pour qu'elle prenne automatiquement toute la moitié gauche de l'écran. Déplacez la fenêtre vers le bord droit de l'écran pour qu'elle prenne automatiquement toute la moitié droite de l'écran. Déplacez la fenêtre vers le bord supérieur de l'écran pour que la fenêtre s'affiche sur tout l'écran.

Sous Windows 7 et Windows Vista, les utilisateurs peuvent placer des gadgets sur le Bureau. Les gadgets sont de petites applications, telles que des jeux, des pense-bêtes, un calendrier ou une horloge. L'illustration 3 montre les gadgets météo, calendrier et horloge de Windows 7. Vous pouvez ancrer ou positionner les gadgets sur les bords et les angles du Bureau, et les aligner les uns par rapport aux autres.

Remarque : Microsoft a cessé de prendre en charge la fonction Gadgets.

Pour ajouter des gadgets sous Windows 7 ou Windows Vista, procédez comme suit :

Étape 1. Cliquez avec le bouton droit sur le Bureau et sélectionnez Gadgets.

Étape 2. Faites glisser le gadget depuis le menu vers le Bureau, ou double-cliquez sur le gadget pour l'ajouter au Bureau ou encore, cliquez avec le bouton droit sur le gadget et choisissez Ajouter.

Étape 3. Pour ancrer un gadget, faites-le glisser vers l'emplacement de votre choix sur le Bureau. Le gadget s'aligne automatiquement sur les bords de l'écran et les autres gadgets.

Sous Windows Vista, vous pouvez également personnaliser la fonction Volet (voir l'illustration 4). Le Volet apparaît sur le Bureau et regroupe les gadgets.
