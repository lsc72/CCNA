Centre de maintenance

Le Centre de maintenance Windows sous Windows 8 et Windows 7 désigne un emplacement centralisé permettant de consulter les alertes et contribuant au bon fonctionnement de Windows. Il est constitué des sections Sécurité et Maintenance. Les messages importants sont surlignés en rouge. Ces problèmes doivent être résolus rapidement. Il peut par exemple s’agir d'un pare-feu désactivé ou mal paramétré, voir l'illustration 1. Les éléments en jaune indiquent des tâches requérant votre attention. Il s'agit par exemple d'une recommandation de créer une sauvegarde, voir l'illustration 2. Le Centre de maintenance propose plusieurs utilitaires :

  • Modifier les paramètres du Centre de maintenance - activation/désactivation des messages de sécurité et des messages de maintenance.

  • Modifier les paramètres de contrôle de compte d'utilisateur - configuration du contrôle de compte d'utilisateur.

  • Afficher les messages archivés - voir les messages archivés concernant les précédents problèmes de l'ordinateur.

  • Afficher les informations de performances - voir et évaluer les performances des composants système.

Pour accéder au Centre de maintenance sous Windows 7 et Windows 8, sélectionnez :

Panneau de configuration > Centre de maintenance

Remarque : Windows 8.0 et Windows 7 incluent la fonction Afficher les informations de performances dans le Centre de maintenance. Microsoft a supprimé cet utilitaire sous Windows 8.1.

Sous Windows Vista, il n’y a pas de Centre de maintenance. Le Centre de sécurité de Windows Vista fournit uniquement les fonctionnalités de sécurité du Centre de maintenance. Il affiche en effet des informations concernant le pare-feu, les mises à jour automatiques, la protection contre les programmes malveillants, les paramètres de sécurité Internet et le contrôle de compte d'utilisateur identifiées par un code-couleur, voir l'illustration 3.

Pour accéder au Centre de sécurité sous Windows Vista, sélectionnez :

Panneau de configuration > Centre de sécurité
