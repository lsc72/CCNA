Numéros de port

Les protocoles TCP et UDP utilisent des numéros de port source et de destination pour suivre les conversations des applications. Le numéro de port source est associé à l'application d'origine sur le périphérique local. Le numéro de port de destination est associé à l'application de destination sur le périphérique distant.

Le numéro de port source est généré de manière dynamique par le périphérique émetteur. Ce processus autorise plusieurs conversations simultanées pour la même application. Par exemple, si vous utilisez un navigateur Web, vous pouvez ouvrir plusieurs onglets à la fois. Le numéro de port de destination est 80 pour le trafic Web et 443 pour le trafic Web sécurisé. Le port source est toutefois différent pour chaque onglet ouvert, ce qui permet à l'ordinateur de savoir dans quel onglet ouvrir le contenu Web. De même, d'autres applications réseau comme la messagerie et le transfert de fichiers ont leurs propres numéros de port attribués.
