Supports réseau

La communication à travers un réseau s'effectue sur un support (média). Celui-ci fournit le canal via lequel le message se déplace de la source à la destination. En présence de supports multiples, on parle de « multimédia ».

Les périphériques réseau sont reliés entre eux via différents supports. Dans l'illustration 1, vous pouvez découvrir les différents supports :

  • Câblage en cuivre - Utilise les signaux électriques pour transmettre les données entre les périphériques.

  • Câblage en fibre optique - Utilise de la fibre de verre ou plastique pour transmettre des informations sous forme d'impulsions lumineuses.

  • Connexion sans fil - Utilise les signaux radio, la technologie à infrarouge ou les transmissions satellites.

Dans l'illustration 2, vous retrouverez les icônes qui seront utilisées tout au long de ce cours pour représenter les différents types de supports réseau. Les réseaux locaux (LAN), les réseaux étendus (WAN) et les réseaux sans fil sont abordés dans la rubrique suivante. Le nuage (cloud en anglais) est utilisé tout au long de ce cours pour représenter les connexions à Internet. Internet est souvent utilisé comme moyen de communication entre un réseau et un autre.
