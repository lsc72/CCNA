The figure on this page is a table that describes the following OSI reference model layers. Layer Name: Application, Layer Number: 7, Description: Responsible for network services to applications. Layer Name: Presentation, layer Number: 6, description: Transforms data formats to provide a standard interface for the application layer. Layer Name: Session, Layer Number: 5, Description: Establishes, manages, and terminates the connections between the local and remote application. Layer Name: Transport, Layer Number: 4, Description: Provides reliable transport and flow control across a network. Layer Name: Network, Layer Number: 3, description: Responsible for logical addressing and the domain of routing. Layer Name: Data Link, layer Number: 2, description: Provides physical addressing and media access procedures. Layer Name: Physical, Layer Number: 1, Description: Defines all the electrical and physical specifications for devices.
