Messagerie Internet

Contrairement aux e-mails locaux où le serveur est géré par un administrateur interne, la messagerie Internet fait référence aux services de messagerie hébergés ailleurs sur Internet et gérés par une équipe d'administrateurs externes.

Comme de plus en plus de personnes commençaient à utiliser les e-mails, il est devenu nécessaire de créer un service de messagerie pour les personnes disposant de peu de connaissances techniques. Certaines entreprises déploient, hébergent et gèrent ce service, laissant aux utilisateurs le soin de gérer leurs messages personnels. Les services de messagerie Internet comportent généralement une interface Web qui permet aux utilisateurs d'accéder à leur messagerie depuis n'importe quel navigateur Web. Outre l'interface Web, les entreprises mettent souvent à disposition une application de messagerie qui peut être téléchargée et installée sur les appareils mobiles. Les applications de messagerie électronique offrent une meilleure expérience utilisateur que l'interface Web.

Les fabricants d'appareils mobiles ajoutent généralement des applications de messagerie à leur système d'exploitation. Ces applications permettent souvent de configurer plusieurs services de messagerie Internet et de les utiliser sans client de messagerie spécifique. Voici quelques-unes des principales options pour accéder à un compte de messagerie Internet :

  • Interface Web

  • Client de messagerie pour ordinateur de bureau avec interface graphique, comme Mail, Outlook, Windows Live Mail et Thunderbird

  • Applications d'e-mail mobiles, y compris Gmail et Yahoo!

  • Applications d'e-mail mobiles du système d'exploitation, comme iOS Mail
