Développement d'applications mobiles

Les systèmes d'exploitation mobiles ne sont pas de simples produits qui fonctionnent seuls. Ce sont des plates-formes sur lesquelles d'autres produits peuvent être créés et diffusés en vue d'être utilisés sur ce système d'exploitation. Ces produits sont par exemple des applications mobiles, appelées aussi simplement applications ou « apps » en anglais. Les applications sont des programmes créés pour effectuer des tâches spécifiques sur les appareils mobiles. Voici quelques exemples d'applications que l'on retrouve généralement sur les appareils mobiles : calendrier, cartes, notes et messagerie électronique.

Dans l'écosystème iOS, les applications étaient initialement conçues par Apple. De même, Google concevait à l'origine les applications Android. Lorsqu'il est apparu que ces systèmes d'exploitation étaient en réalité des plates-formes logicielles, Apple et Google ont tous les deux publié leur propre kit de développement logiciel (SDK). Ce kit contient un certain nombre d'outils logiciels conçus pour créer des programmes externes destinés à un package logiciel spécifique.

Les développeurs qui souhaitent créer des applications Apple iOS doivent télécharger et installer Xcode, l'environnement de développement intégré officiel (IDE) d'Apple. XCode, voir l'illustration 1, peut être téléchargé gratuitement. Il permet aux développeurs de créer et de tester leurs applications iOS dans un simulateur d'iPhone. Le chargement et l'exécution des applications actives sur l'appareil iOS réel nécessitent le paiement d'un abonnement annuel au programme développeur pour iOS. Xcode contient également un programme de débogage, des bibliothèques, un simulateur d'appareil, de la documentation, des exemples de code et des didacticiels.

Le SDK de Google pour Android propose également plusieurs des éléments mentionnés pour Xcode, ainsi qu'un certain nombre de didacticiels. Le SDK Android prend en charge plusieurs plates-formes de développement, dont les ordinateurs Linux, Mac OS X 10.5.8 ou versions ultérieures et Windows XP et versions ultérieures. Android Studio est un IDE créé par Google pour le développement sur Android.
