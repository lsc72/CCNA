Fonctionnalités pratiques

Les appareils mobiles modernes sont essentiellement de petits ordinateurs. Par conséquent, un certain nombre de fonctionnalités pratiques ont été créées pour les smartphones modernes. Voici quelques-unes de ces fonctionnalités :

Appel Wi-Fi

Au lieu d'utiliser le réseau de l'opérateur de téléphonie mobile, les smartphones modernes peuvent utiliser Internet pour acheminer les appels vocaux grâce à un point d'accès Wi-Fi. On parle d'appel Wi-Fi. Un réseau Wi-Fi connecté à Internet est généralement disponible dans des lieux, tels que des cafés, des lieux de travail, des bibliothèques ou simplement des domiciles. Le téléphone peut faire transiter les appels vocaux par le point d'accès Wi-Fi local. Si aucun point d'accès Wi-Fi n'est à portée, le téléphone utilise le réseau de l'opérateur de téléphonie mobile pour acheminer les appels vocaux.

La fonction d'appel Wi-Fi est très utile dans les zones où la couverture mobile est médiocre, car elle utilise le point d'accès Wi-Fi local pour combler les lacunes. Le point d'accès Wi-Fi doit pouvoir garantir un débit Internet d'au moins 1 Mbit/s pour un appel de bonne qualité. Lorsque la fonction d'appel Wi-Fi est activée et en service pendant l'appel vocal, le téléphone affiche « Wi-Fi » en regard du nom de l'opérateur.

Pour activer la fonction Appel Wi-Fi sur les appareils Android, voir l'illustration 1, sélectionnez :

Paramètres > Plus (dans la section Sans fil et réseaux) > Appel Wi-Fi > Appuyez sur l'interrupteur pour l'activer

Pour activer la fonction Appel Wi-Fi sur les appareils iOS, voir l'illustration 2, sélectionnez :

Paramètres > Téléphone et activez Appel Wi-Fi

Remarque : tous les opérateurs de téléphonie mobile n'autorisent pas la fonction d'appel Wi-Fi. Si vous ne parvenez pas à l'activer sur votre téléphone, votre opérateur ou votre appareil mobile ne prend probablement pas cette fonction en charge.

Paiements mobiles

Les paiements mobiles, voir l'illustration 3, sont tous les paiements effectués via un téléphone mobile. Voici quelques modes de paiement mobile :

  • Paiements avec transaction basées sur SMS : le client envoie à un numéro de téléphone spécifique de l'opérateur un SMS contenant une demande de paiement. Le vendeur est informé que le paiement a été reçu et qu'il peut remettre le bien au client. Le montant est ensuite ajouté à la facture de téléphonie mobile du client. Les points faibles de cette méthode sont la lenteur, le manque de fiabilité et le faible niveau de sécurité.

  • Facturation mobile directe : avec l'option de facturation mobile lors du paiement, l'utilisateur s'identifie (généralement via une authentification à deux facteurs) et autorise l'ajout du montant à sa facture de services mobiles. Cette solution est très répandue en Asie et offre plusieurs avantages : sécurité, commodité et absence de carte de paiement ou de carte de crédit.

  • Paiement Web mobile : le client utilise le Web ou des applications dédiées pour effectuer la transaction. Cette méthode s'appuie sur le protocole WAP (Wireless Application Protocol) et requiert généralement l'utilisation d'une carte de crédit ou d'une solution de paiement en ligne préalablement enregistrée, comme PayPal.

  • NFC sans contact : cette méthode est principalement utilisée dans les transactions physiques en magasin. Le client paie le bien ou le service en approchant le téléphone du système de paiement. Sur la base d'un identifiant unique, le paiement est directement facturé sur un compte prépayé ou sur le compte bancaire. La technologie NFC est également utilisée dans les services de transport en commun, le stationnement, ainsi que de nombreux domaines grand public.

Réseau privé virtuel

Un réseau privé virtuel (VPN) est un réseau privé qui a recours à un réseau public, par exemple Internet, pour connecter entre eux des sites ou des utilisateurs distants. Au lieu d'utiliser une ligne louée dédiée, le VPN se sert de connexions virtuelles routées via Internet, entre le réseau privé de l'entreprise et l'employé ou le site distant.

De nombreuses entreprises créent leurs propres VPN pour répondre aux besoins des employés et des bureaux distants. Avec la multiplication des appareils mobiles, l'apparition des clients VPN sur les smartphones et les tablettes était une évolution naturelle.

Lorsque la connexion VPN est établie entre le client et le serveur, le client accède au réseau qui se trouve derrière le serveur comme s'il était directement connecté à ce réseau. Comme les protocoles VPN permettent également le chiffrement des données, la communication entre le client et le serveur est sécurisée, et donc intéressante pour les entreprises.

Pour créer une connexion VPN sur Android, voir l'illustration 4, sélectionnez :

Paramètres > Plus (dans la section Sans fil et réseaux) > VPN > appuyez sur le signe + pour ajouter une connexion VPN

Pour créer une nouvelle connexion VPN sur iOS (illustration 5), sélectionnez :

Réglages > Général > VPN > Ajouter une configuration VPN

Lorsque les informations du VPN ont été ajoutées sur l'appareil, celui-ci doit être démarré avant de pouvoir envoyer et recevoir du trafic.

Pour démarrer une connexion VPN sur Android (voir l'illustration 6), sélectionnez :

Paramètres > Général > VPN > sélectionnez la connexion VPN souhaitée > entrez le nom d'utilisateur et le mot de passe > appuyez sur CONNECTER

Pour démarrer une connexion VPN sur iOS (illustration 7), sélectionnez :

Réglages > appuyez sur l'interrupteur VPN
