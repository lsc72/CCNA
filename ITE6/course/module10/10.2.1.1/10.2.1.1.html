Brève présentation du verrouillage par code secret

Les smartphones, tablettes et autres appareils mobiles contiennent souvent des données sensibles. Lorsqu'un tel appareil est perdu, la personne qui le retrouve a donc accès aux contacts, aux SMS et aux comptes Web de l'utilisateur. L'utilisation d'un verrouillage par code secret est l'une des méthodes permettant d'éviter le vol d'informations confidentielles stockées dans des appareils mobiles. Le verrouillage par code secret permet de sécuriser un appareil mobile et de le mettre en veille. Le verrouillage peut également être différé et s'activer après une durée de veille spécifique. En principe, pour mettre un appareil mobile en veille il faut appuyer brièvement sur le bouton d'alimentation principal. Il est également possible de régler l'appareil pour qu'il se mette en veille après une certaine période d'inactivité.

De nombreuses méthodes de verrouillage par code secret sont disponibles (voir l'illustration 1). Certaines sont plus difficiles à contourner que d'autres. Le code secret doit être fourni chaque fois que l'utilisateur allume l'appareil ou le sort du mode veille. Voici les méthodes de verrouillage par code secret les plus courantes :

  • Aucun : supprime tout type de verrouillage par code sur cet appareil.

  • Glissement : l'utilisateur doit faire glisser une icône, par exemple un cadenas ou une flèche, pour déverrouiller l'appareil. C'est la méthode la moins sûre.

  • Modèle : l'utilisateur doit dessiner un schéma sur l'écran. Pour déverrouiller l'appareil, le même schéma doit être dessiné sur l'écran.

  • Code PIN : un code PIN est utilisé pour protéger l'appareil. Le code PIN est une suite de chiffres. Lorsque le code PIN saisi est correct, l'appareil se déverrouille.

  • Mot de passe : un mot de passe est utilisé pour protéger l'appareil. Cette méthode est la moins pratique, en particulier si le mot de passe est compliqué ou long, mais c'est aussi celle qui peut être la plus sûre.

  • Code simple : disponible uniquement sur iOS. Lorsque cette option est activée, il faut taper un code secret à 4 chiffres pour déverrouiller l'appareil. Si cette option est désactivée, des mots de passe plus complexes (contenant des caractères spéciaux, des chiffres et des lettres) peuvent être utilisés.

  • Touch ID : appareils iOS uniquement. Depuis l'iPhone 5S, il est également possible de déverrouiller un appareil iOS avec ses empreintes digitales. Le bouton principal de certains iPhone contient un lecteur d'empreintes qui peut être utilisé pour déverrouiller l'appareil, autoriser les paiements et accéder aux applications. Bien que Google ait annoncé une fonctionnalité similaire, le déverrouillage par empreintes digitales n'était pas encore très répandu sur les appareils Android au moment de la rédaction de ce chapitre.

Pour définir un code secret sur un appareil Android, procédez comme suit :

Paramètres > Sécurité > Verrouillage de l'écran > Choisissez le type de code à utiliser, puis configurez les autres paramètres de sécurité.

Une fois le code secret défini, vous devez l'entrer chaque fois que vous allumez l'appareil ou que vous le sortez du mode veille.

Android 5.0 (Lollipop) offre plusieurs options de déverrouillage supplémentaires, voir l'illustration 2. Regroupées dans la fonction Smart Lock, ces options de déverrouillage d'Android sont les suivantes :

  • Appareils fiables : cette fonction déverrouille le téléphone ou la tablette Android à chaque fois qu'il est jumelé à un périphérique Bluetooth spécifique ou à chaque passage près de l'étiquette NFC choisie par l'utilisateur. Par exemple, il peut être pratique de pouvoir laisser le téléphone ou la tablette Android déverrouillée une fois qu'elle est jumelée au Bluetooth de la voiture.

  • Lieux de confiance : grâce à la puce GPS de l'appareil, Android permet à l'utilisateur de définir son lieu de travail et son domicile. L'appareil ne demande plus à être déverrouillé lorsqu'il se trouve à un de ces endroits.

  • Visage de confiance : cette fonction permet à Android de se déverrouiller en fonction d'une image préalablement enregistrée du visage de l'utilisateur.

  • Voix de confiance : cette fonction repose sur la signature vocale unique de l'utilisateur pour déverrouiller l'appareil Android lorsqu'il dit « OK Google ».

  • Détection de l'appareil porté : cette fonctionnalité utilise l'accéléromètre de l'appareil pour détecter si l'utilisateur l'a en sa possession. Lorsqu'il détecte un mouvement spécifique, Android suppose que l'appareil est dans la main de l'utilisateur, dans sa poche ou dans un sac et qu'il n'a pas besoin d'être déverrouillé tant qu'il n'est pas posé ailleurs par l'utilisateur.

Pour activer Smart Lock sur un appareil Android, procédez comme suit :

Paramètres > Sécurité > Smart Lock > > Choisissez le type de code à utiliser, puis configurez les autres paramètres de sécurité.

Pour paramétrer un code secret sur un appareil iOS, procédez comme suit :

Réglages > Touch ID et code > Appuyez sur Activer le code. Entrez un numéro à quatre chiffres (voir l'illustration 3). Entrez ce même code une seconde fois.

Pour utiliser Touch ID, l'utilisateur doit enregistrer au moins une empreinte digitale. Il peut en enregistrer jusqu'à cinq. La configuration et le comportement de Touch ID peuvent également être définis dans l'écran décrit ci-dessus.

Remarque : pour améliorer la sécurité, certains services forcent l'utilisateur à créer un verrouillage par code secret avant de pouvoir être activés, par exemple les comptes VPN et les comptes Microsoft Exchange.
