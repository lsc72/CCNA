Éléments de l'écran d'accueil

L'interface d'iOS fonctionne en grande partie comme l'interface d'Android. Les écrans sont utilisés pour organiser les applications. Vous devez appuyer sur les applications pour les exécuter. Il existe cependant des différences notables entre ces deux interfaces :

  • Aucune icône de navigation : sur iOS, un bouton matériel remplace les icônes de navigation d'Android.

  • Pas de widgets : sur iOS, les widgets ne sont pas utilisés.

  • Aucun raccourci d'application : sur iOS, toutes les applications présentes sur l'écran d'accueil sont de véritables applications et non des raccourcis.

Bouton principal

Contrairement à Android, les appareils exécutant iOS n'utilisent pas d'icônes de navigation. Un simple bouton physique, le bouton principal, permet d'effectuer la plupart des fonctions des boutons de navigation d'Android. Le bouton principal se trouve en bas de l'appareil, voir l'illustration 1. Le bouton principal exécute les fonctions suivantes :

  • Sortir l'appareil du mode veille : lorsque l'écran de l'appareil est éteint, appuyez sur le bouton principal pour le rallumer.

  • Revenir à l'écran d'accueil : appuyez sur le bouton principal quand une application est en cours d'utilisation pour revenir au dernier écran d'accueil utilisé.

  • Ouvrir la barre du mode multitâche : appuyez deux fois de suite sur le bouton principal pour ouvrir cette barre. Celle-ci affiche les applications récemment ouvertes.

  • Démarrer Siri ou la commande vocale : appuyez sur le bouton principal et maintenez-le enfoncé pour démarrer Siri ou la commande vocale. Siri est un logiciel permettant un contrôle vocal avancé. Sur les appareils ne disposant pas de Siri, les commandes vocales de base peuvent être utilisées de la même manière.

  • Ouvrir les paramètres audio : appuyez deux fois de suite sur le bouton principal lorsque l'écran est verrouillé pour ouvrir les paramètres audio.

Centre de notifications

Les appareils iOS ont un centre de notifications qui regroupe toutes les alertes en un seul emplacement, voir l'illustration 2. Pour ouvrir la zone de notification sur un appareil iOS, appuyez sur le haut de l'écran et faites glisser votre doigt vers le bas. Les actions suivantes peuvent être effectuées depuis le centre de notifications :

  • Répondre à une alerte en appuyant dessus.

  • Supprimer une alerte en appuyant sur l'icône X, puis en sélectionnant Effacer.

Pour modifier les options des notifications application par application, sélectionnez :

Réglages > Notifications

Paramètres courants

Les appareils iOS permettent à l'utilisateur d'accéder rapidement aux réglages et aux commutateurs les plus utilisés, même si l'appareil est verrouillé. Pour accéder au menu des réglages les plus utilisés, voir l'illustration 3, balayez depuis le bas de l'écran. Dans l'écran des réglages, l'utilisateur peut :

  • Modifier les réglages les plus utilisés, comme le mode Avion, le Wi-Fi, le Bluetooth, le mode Ne pas déranger et le verrouillage de la rotation de l'écran

  • Régler la luminosité de l'écran

  • Contrôler le lecteur de musique

  • Accéder à Airdrop

  • Accéder à la lampe torche, à l'horloge, au calendrier et à l'appareil photo

Spotlight (outil de recherche)

Depuis n'importe quel écran de l'appareil iOS, appuyez sur l'écran et balayez vers le bas pour ouvrir la zone de recherche Spotlight. Toutes les parties de l'écran fonctionnent, mis à part le haut et le bas. Lorsque le champ de recherche Spotlight est affiché, tapez ce que vous recherchez. Spotlight présente des suggestions provenant de plusieurs sources (Internet, iTunes, App Store, heures des séances de cinéma, lieux à proximité, etc.). Spotlight met aussi automatiquement à jour les résultats en fonction de ce que vous tapez, voir l'illustration 4.
