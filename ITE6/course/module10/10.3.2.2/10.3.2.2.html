Jumelage Bluetooth

Le jumelage Bluetooth consiste à établir une connexion entre deux appareils Bluetooth pour partager des ressources. Lors de cette association, les fonctions radio Bluetooth doivent être activées et l'un des appareils lance une recherche pour détecter les périphériques à portée. Les autres appareils doivent être détectables (c'est-à-dire, visibles). Lorsqu'un appareil Bluetooth est détectable, il transmet les informations suivantes lorsqu'elles sont demandées par un autre appareil Bluetooth :

  • Nom

  • Classe d'émetteur Bluetooth

  • Services utilisables par l'appareil

  • Informations techniques (par exemple, fonctionnalités et compatibilité)

Lors du processus de jumelage, un code PIN peut également être demandé afin d'authentifier le processus de jumelage, voir l'illustration 1. Le code PIN est la plupart du temps un nombre, mais peut également être un code alphanumérique ou une clé d'accès. Le code PIN est enregistré lorsque le jumelage est utilisé. Par conséquent, il ne doit être saisi qu'une seule fois. C'est très pratique, par exemple lorsqu'on utilise un casque avec un smartphone : le jumelage s'effectue automatiquement dès la mise sous tension du casque et lorsqu'il est à portée.

Pour associer un appareil Bluetooth avec un appareil Android, procédez comme suit :

Étape 1. Suivez les instructions relatives à votre appareil pour le rendre détectable.

Étape 2. Suivez les instructions relatives à votre appareil pour trouver le code PIN de connexion.

Étape 3. Sélectionnez Paramètres > Bluetooth (dans la section Sans fil et réseaux).

Étape 4. Appuyez sur l'interrupteur Bluetooth pour l'activer.

Étape 5. Attendez qu'Android repère le périphérique Bluetooth en mode détectable.

Étape 6. Appuyez sur l'un des appareils détectés pour le sélectionner.

Étape 7. Saisissez le code PIN.

Pour associer un appareil Bluetooth avec un appareil iOS, procédez comme suit :

Étape 1. Suivez les instructions relatives à votre appareil pour le rendre détectable.

Étape 2. Suivez les instructions relatives à votre appareil pour trouver le code PIN de connexion.

Étape 3. Sélectionnez Réglages > Bluetooth.

Étape 4. Appuyez sur Bluetooth pour l'activer (voir l'illustration 2).

Étape 5. Appuyez sur l'un des appareils détectés pour le sélectionner.

Étape 6. Saisissez le code PIN.

Pour certains appareils, ce processus peut être légèrement différent. Les Figures 1 et 3 présentent un clavier Bluetooth Microsoft jumelé avec Android et iOS, respectivement. Notez que le clavier demande de saisir un code PIN avant de terminer le processus de jumelage.
