Présentation des systèmes d'exploitation Linux et OS X

Unix est un système d'exploitation non propriétaire basé sur le langage de programmation C, avec une interface de commande utilisateur. Certains systèmes d'exploitation répandus sont basés sur UNIX, comme Linux, OS X, Android et iOS.

Linux

Les systèmes d'exploitation Linux sont utilisés dans presque toutes les plates-formes : systèmes embarqués, montres ey autres objets connectés, téléphones portables, miniportables, ordinateurs, serveurs, superordinateurs... Bien que Linux soit de plus en plus largement utilisé, Android, une version modifiée de Linux, est sans aucun doute responsable de la diffusion de ce système d'exploitation sur le marché grand public.

OS X (anciennement Mac OS X)

OS X, anciennement Mac OS X, est le système d'exploitation des ordinateurs Macintosh. OS X est optimisé pour le matériel informatique Macintosh et fonctionne de manière fluide avec les autres appareils Apple, comme les iPhone.

L'illustration 2 présente le bureau d'OS X 10.10 Yosemite

Remarque : ce chapitre porte sur Ubuntu Linux 14.04 LTS (Trusty Tahr) et Apple OS X 10.10 (Yosemite).

Un outil spécial intégré à OS X permet de créer des configurations d'installation uniques. Cet outil, Netboot, peut démarrer plusieurs ordinateurs OS X à distance. Une fois les ordinateurs redémarrés, tout le contenu de la session est perdu. Cet outil peut servir par exemple dans une salle de classe pour réinitialiser les ordinateurs à la fin du cours ou à la fin de la journée. Il peut aussi gérer l'installation d'un système d'exploitation ou d'un programme sur plusieurs ordinateurs à la fois, ce qui peut notamment permettre à une entreprise d'appliquer une image ou une mise à niveau sur tous les ordinateurs.
