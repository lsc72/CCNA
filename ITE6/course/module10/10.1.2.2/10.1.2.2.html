Gestion des applications, des widgets et des dossiers

Applications

Chaque écran d'accueil est constitué d'une grille où peuvent être placées les applications. Pour déplacer une application, procédez comme suit :

Étape 1. Appuyez sur l'application et maintenez la pression sur l'écran.

Étape 2. Faites glisser l'application vers une zone vide d'un des écrans d'accueil.

Étape 3. Relâchez l'application.

Pour supprimer une application d'un écran d'accueil, procédez comme suit :

Étape 1. Appuyez sur l'application et maintenez la pression sur l'écran.

Étape 2. Faites-la glisser vers X Supprimer en haut de l'écran.

Étape 3. Relâchez l'application.

Pour exécuter une application, appuyez sur l'icône correspondante. Une fois l'application lancée, certaines options peuvent généralement être configurées via le bouton Menu.

Il existe 3 moyens de fermer une application :

  • Appuyer sur le bouton Retour jusqu'à ce que l'écran d'accueil s'affiche En général, dans ce cas, un message vous proposant de quitter l'application s'affiche.

  • Appuyer sur le bouton Accueil.

  • Sélectionner l'option Quitter dans le menu de l'application.

Widgets

Les widgets sont des programmes (ou des parties de programmes) affichant des informations directement sur l'écran d'accueil. Par exemple, le widget météo peut être placé sur l'écran d'accueil pour afficher la météo. Souvent, une pression sur un widget lance l'application associée. Par exemple, appuyer sur un widget météo peut ouvrir l'application de prévisions météorologiques en plein écran, et offrir plus d'informations sur les conditions météo. Les widgets sont utiles car ils permettent un accès rapide aux fonctions et informations les plus couramment utilisées. Voici quelques exemples de widgets courants :

  • Horloge : affiche une grande horloge personnalisable.

  • Météo : affiche les conditions météorologiques actuelles pour une ou plusieurs villes.

  • Marche/arrêt du Wi-Fi : permet à l'utilisateur d'activer/de désactiver le Wi-Fi rapidement, sans avoir à accéder au menu Paramètres.

  • Gestion de l'alimentation : affiche plusieurs widgets, par exemple pour activer/désactiver le Wi-Fi, le Bluetooth et le vibreur.

Il existe de nombreux autres widgets permettant de personnaliser les écrans d'accueil d'Android. Consultez la documentation des programmes ou Google Play pour savoir quelles applications proposent des widgets. Pour ajouter un widget à l'écran d'accueil, procédez comme suit :

Étape 1. Appuyez sur une zone vide de l'écran d'accueil et maintenez la pression sur l'écran.

Étape 2. Appuyez sur Widgets.

Étape 3. Recherchez le widget souhaité dans la liste.

Étape 4. Appuyez sur le widget souhaité et maintenez la pression, puis faites-le glisser vers l'écran d'accueil.

Dossiers

Sur certains appareils mobiles, plusieurs applications peuvent être regroupées dans un dossier, à des fins d'organisation. Si les dossiers ne sont pas disponibles sur un appareil mobile, il est possible de télécharger une application proposant cette fonctionnalité. Les applications peuvent être regroupées exactement comme vous le souhaitez.

Pour créer un dossier sur un appareil Android, procédez comme suit :

Étape 1. Appuyez sur une application dans l'un des écrans d'accueil et maintenez la pression sur l'écran.

Étape 2. Faites glisser l'application sur l'autre application que vous voulez placer dans le même dossier.

Étape 3. Relâchez l'application.

Appuyez sur un dossier pour l'ouvrir. Appuyez sur une application située dans un dossier pour l'ouvrir. Pour renommer un dossier, appuyez sur le dossier, puis sur son nom, et tapez le nouveau nom du dossier. Pour fermer un dossier, appuyez sur l'écran en dehors du dossier, ou appuyez sur le bouton Retour ou Accueil. Vous pouvez déplacer les dossiers sur les écrans d'accueil de la même manière que les applications.

Pour retirer une application d'un dossier, procédez comme suit :

Étape 1. Ouvrez le dossier.

Étape 2. Appuyez sur l'application à supprimer et maintenez la pression sur l'écran.

Étape 3. Faites glisser l'application vers une zone vide de l'écran d'accueil.

Étape 4. Relâchez l'application.

Sur les appareils Android, si un dossier contient deux applications et que l'une d'elles est retirée du dossier, ce dossier est supprimé, mais l'application restante remplace le dossier sur l'écran d'accueil.

Icône Toutes les applications

L'icône Toutes les applications ouvre un écran Toutes les applications. Il contient toutes les applications installées sur l'appareil. Voici quelques-unes des tâches les plus courantes pouvant être effectuées depuis cet écran :

  • Lancer une application : appuyez sur une application pour l'ouvrir.

  • Placer une application sur un écran d'accueil : appuyez sur l'application et maintenez la pression. L'écran d'accueil s'affiche. Relâchez l'application sur une zone vide de l'un des écrans d'accueil.

  • Désinstaller une application : appuyez dessus et maintenez la pression. Faites glisser l'application vers la corbeille ou l'icône X.

  • Accéder au Play Store : appuyez sur l'icône Menu et sélectionnez Google Play.
