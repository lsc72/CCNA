Sauvegarde à distance

Les données des appareils mobiles peuvent être perdues à cause de défaillances matérielles ou de la perte ou du vol de l'appareil. Il est donc important de sauvegarder régulièrement les données afin de pouvoir les récupérer si nécessaire. Cependant, l'espace de stockage des appareils mobiles est souvent limité et inamovible. Pour contourner ces restrictions, il est possible d'effectuer des sauvegardes à distance. La sauvegarde à distance consiste à copier les données d'un appareil sur un stockage cloud à l'aide d'une application dédiée. Lorsque les données doivent être restaurées, utilisez l'application de sauvegarde et accédez au site Web pour récupérer les données.

Pour les appareils Android, les éléments suivants sont automatiquement sauvegardés à distance :

  • Agenda

  • E-mails

  • Contacts

Google garde également une trace de toutes les applications et de tous les contenus téléchargés, ce qui permet de les télécharger de nouveau. De nombreuses applications peuvent également sauvegarder à distance les contenus non gérés par Google. Vous pourrez trouver des applications de sauvegarde correspondant à vos besoins dans le Google Play Store.

Les utilisateurs d'iOS ont droit à 5 Go de stockage gratuit, comme illustré ci-contre. Ils peuvent étendre cette capacité via un abonnement annuel. Voici les éléments qui peuvent utiliser iCloud comme emplacement de sauvegarde :

  • Agenda

  • E-mails

  • Contacts

  • Contenu acheté sur l'App Store d'Apple (non compris dans les 5 Go d'espace de stockage gratuit)

  • Photos prises avec l'appareil

  • Paramètres configurés sur l'appareil

  • Données accumulées par les applications actives

  • Icônes et organisation des écrans

  • SMS et MMS

  • Sonneries
