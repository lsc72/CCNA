Logiciels open source et logiciels propriétaires

Comme les ordinateurs de bureau et les ordinateurs portables, les appareils mobiles utilisent un système d'exploitation pour exécuter les logiciels. Ce chapitre porte sur les deux systèmes d'exploitation mobiles les plus courants : Android et iOS. Android est développé par Google et iOS est développé par Apple.

Pour que les utilisateurs puissent analyser et modifier le logiciel, ils doivent disposer du code source. Le code source est la séquence d'instructions écrite de façon compréhensible par l'homme avant d'être transformée en langage machine (zéro et un). Le code source est un élément important des logiciels gratuits, car il permet aux utilisateurs de l'analyser et de le modifier par la suite. Lorsque le développeur choisit de mettre le code source à disposition, le logiciel est dit « open source ». Si le code source du programme n'est pas publié, le logiciel est dit propriétaire.

Android est un système d'exploitation open source développé par Google. iOS est un système d'exploitation fermé, ou propriétaire, développé par Apple.

Lancé en 2008 sur le HTC Dream, le système d'exploitation Android a été adapté pour pouvoir être utilisé sur différents appareils. Ce système d'exploitation étant ouvert et personnalisable, les programmeurs peuvent l'utiliser sur des ordinateurs portables, des télévisions connectées ou des livres numériques. Android a même été installé sur des caméras, des GPS et des lecteurs multimédias portables. L'illustration 1 montre Android exécuté sur une tablette.

Apple iOS est sorti en 2007 sur le premier iPhone, mais son code source n'a pas été rendu public. Toute modification, copie ou redistribution d'iOS nécessite une autorisation d'Apple. L'illustration 2 montre iOS exécuté sur un iPhone.

Ce chapitre porte sur Android 5.0.1 (Lollipop) et iOS 8.4, car il s'agissait des dernières versions disponibles au moment de la rédaction.
