Gestion des applications et des dossiers

Applications

Les applications (appelées « apps ») et dossiers iOS fonctionnent de la même manière que sur Android. Au lieu d'utiliser un bouton Toutes les applications pour afficher les applications installées sur l'appareil, celles-ci sont affichées sur les écrans d'accueil. Les applications peuvent être désinstallées de l'appareil puis réinstallées à l'aide d'iTunes.

Chaque écran d'accueil est constitué d'une grille où peuvent être placées les applications. Pour déplacer une application, procédez comme suit :

Étape 1. Appuyez sur une application et maintenez la pression jusqu'à ce que l'icône bouge légèrement.

Étape 2. Faites glisser l'application vers une zone vide d'un des écrans d'accueil.

Étape 3. Relâchez l'application.

Étape 4. Déplacez les autres applications de votre choix.

Étape 5. Appuyez sur le bouton principal pour valider les modifications.

Pour supprimer une application d'un appareil iOS, procédez comme suit :

Étape 1. Appuyez sur une application et maintenez la pression jusqu'à ce que l'icône bouge légèrement.

Étape 2. Appuyez sur l'icône X qui s'affiche sur l'application.

Étape 3. Supprimez les autres applications de votre choix.

Étape 4. Appuyez sur le bouton principal pour valider les modifications.

Barre du mode multitâche

iOS offre la possibilité d'exécuter plusieurs applications simultanément. Des applications peuvent en effet s'exécuter en arrière-plan alors que vous utilisez une autre application. La barre du mode multitâche vous permet de passer rapidement d'une application à une autre (parmi les applications utilisées récemment), de fermer les applications en cours d'exécution et d'accéder à des paramètres courants. Pour ouvrir la barre du mode multitâche, voir l'illustration 1, appuyez deux fois sur le bouton principal. Lorsque la barre multitâche est visible, iOS affiche également vos contacts favoris et vos contacts récents en haut de l'écran. Les actions suivantes peuvent être effectuées en mode multitâche :

  • Appuyer sur une application pour l'ouvrir.

  • Faire glisser la barre du mode multitâche vers la droite ou vers la gauche pour afficher plus d'applications.

  • Appuyer sur n'importe quel contact favori ou contact récent en haut de l'écran pour y accéder.

  • Faire glisser une application vers le haut pour la forcer à se fermer.

Dossiers

Des dossiers peuvent être créés sur iOS pour organiser les applications. Pour créer un dossier, procédez comme suit :

Étape 1. Appuyez sur une application de l'écran d'accueil et maintenez la pression sur l'écran jusqu'à ce que l'icône bouge légèrement.

Étape 2. Faites glisser l'application sur l'autre application que vous voulez placer dans le même dossier.

Étape 3. Relâchez l'application.

Étape 4. Ajoutez les autres applications de votre choix au dossier.

Étape 5. Appuyez sur le bouton principal pour valider les modifications.

Appuyez sur un dossier pour l'ouvrir. Appuyez sur une application située dans un dossier pour l'ouvrir. Pour renommer un dossier, appuyez sur le dossier, puis sur son nom, et tapez le nouveau nom du dossier. Pour fermer un dossier, appuyez sur l'écran en dehors du dossier, ou appuyez sur le bouton principal. Vous pouvez déplacer les dossiers sur les écrans d'accueil de la même manière que les applications.

Pour retirer une application d'un dossier, procédez comme suit :

Étape 1. Ouvrez le dossier.

Étape 2. Appuyez sur l'application à retirer et maintenez la pression sur l'écran.

Étape 3. Faites glisser l'application vers une zone vide de l'écran d'accueil.

Étape 4. Relâchez l'application.

Pour supprimer un dossier, retirez toutes les applications présentes dans le dossier.

De nombreuses applications iOS peuvent afficher des alertes. Un badge d'alerte s'affiche sous la forme d'une petite icône apparaissant sur l'application, voir l'illustration 2. Cette icône indique le nombre d'alertes de l'application qui doivent être traitées. Cette alerte peut être un appel en absence, le nombre de SMS non lus ou le nombre de mises à jour disponibles. Si une alerte affiche un point d'exclamation, l'application en question rencontre un problème. Les icônes d'alerte des applications contenues dans un dossier sont affichées sur le dossier. Appuyez sur l'application pour consulter les alertes.
