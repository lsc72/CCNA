Documenter les observations, les actions et les résultats

Une fois l'ordinateur réparé, terminez la procédure de dépannage avec le client. Expliquez-lui le problème et la solution, verbalement et par écrit. L'illustration ci-contre indique la marche à suivre une fois la réparation terminée.

Vérifiez avec le client que la solution fonctionne. S'il a le temps, montrez-lui comment vous avez résolu le problème de l'ordinateur. Demandez-lui de tester la solution et de tenter de reproduire le problème. Lorsque le client peut s'assurer que le problème a été résolu, vous pouvez finir de documenter la réparation dans l'ordre de travail et dans votre journal. Incluez les informations suivantes :

  • Description du problème

  • Étapes suivies pour résoudre le problème

  • Composants utilisés pour la réparation
