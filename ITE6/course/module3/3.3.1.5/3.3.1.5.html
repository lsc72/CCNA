Mise à niveau de la mémoire vive

Dans la plupart des cas, l'ajout de mémoire vive permet d'améliorer les performances du système. Avant de mettre à niveau ou de remplacer la mémoire vive, vous devez aux questions indiquées ci-contre.

Remarque : la mémoire doit souvent être installée dans un ordre spécifique sur une carte mère. N'oubliez pas de consulter le guide d'utilisation de la carte mère pour plus d'informations.

Pour mettre à niveau la mémoire vive sur une carte mère, procédez comme suit :

Étape 1. Retirez le module existant en libérant les clips de fixation.

Étape 2. Ôtez le module du socket.

Étape 3. Insérez le nouveau module dans le socket.

Étape 4. Verrouillez les clips de fixation.

Le système détecte si la nouvelle mémoire vive est compatible et si elle est correctement installée. Si le BIOS n'indique pas la présence de la bonne quantité de mémoire vive, vérifiez que la mémoire vive est compatible avec la carte mère et qu'elle est correctement installée.
