Programme de configuration du BIOS

Il peut être nécessaire de modifier les paramètres par défaut du BIOS chaque fois que des composants matériels tels que des modules de mémoire, des périphériques de stockage et des cartes d'extension sont ajoutés ou remplacés. Le programme de configuration du BIOS doit être utilisé pour modifier les paramètres.

Pour entrer dans le programme de configuration du BIOS, appuyez sur la touche ou la séquence de touches appropriée lors du test POST. Cette séquence de touches varie d'un fabricant à l'autre, mais il s'agit souvent de la touche Suppr ou d'une touche de fonction. Par exemple, avec une carte mère ASUS, utilisez la touche Suppr ou la touche de fonction F2 lors du test POST pour entrer dans le programme du BIOS.

Remarque : consultez la documentation de la carte mère afin de connaître la touche ou la combinaison de touches à utiliser sur votre ordinateur.

De nombreuses cartes mères affichent un écran de démarrage lorsque l'ordinateur effectue le processus POST. Cet écran affiche parfois la combinaison de touches du fabricant permettant d'accéder au BIOS.

Bien que les programmes de configuration du BIOS diffèrent d'un fabricant à un autre, ils fournissent tous un accès à des éléments de menu similaires à ceux présentés dans l'illustration ci-contre :

  • Main (Principal) : configuration système de base

  • Advanced (Avancé) : paramètres avancés du système

  • Boot (Démarrage) : options de démarrage des périphériques et ordre de démarrage

  • Security (Sécurité) : paramètres de sécurité

  • Power (Alimentation) : paramètres de configuration avancés de l'alimentation

  • JUSTw00t! : paramètres de tension et d'horloge avancés

  • Exit (Quitter) : options de sortie du BIOS et chargement des paramètres par défaut

Remarque : les écrans de configuration du BIOS dans cette section sont fournis à titre de référence et sont probablement différents des vôtres. Considérez qu'il s'agit juste d'exemples et consultez la documentation du fabricant de votre carte mère.
