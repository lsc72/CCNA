Mode EZ de l'UEFI

Sur les ordinateurs récents, l'UEFI remplace l'interface BIOS. Bien que les écrans UEFI varient d'un fournisseur à un autre, ils offrent tous des fonctionnalités similaires.

L'illustration présente l'utilitaire du BIOS UEFI d'ASUS en mode EZ. Par défaut, l'écran du mode EZ apparaît lorsque vous accédez à l'UEFI. Le mode EZ fournit un aperçu des informations de base du système, affiche la température du processeur et de la carte mère, la tension de sortie du processeur, la vitesse des ventilateurs, les informations relatives à la mémoire et les informations SATA.

Le mode EZ vous permet d'effectuer les opérations suivantes :

  • modifier l'heure et la date ;

  • sélectionner la langue du programme ;

  • utiliser l'Assistant de réglage EZ pour automatiser la création du stockage RAID et pour faciliter l'overclocking du système ;

  • utiliser le réglage de système EZ pour définir le mode de performances du système ;

  • activer ou désactiver le RAID SATA ;

  • sélectionner la séquence de démarrage des périphériques.

Il existe un autre mode, le mode avancé, qui permet de configurer et de gérer les paramètres avancés Pour basculer entre le mode EZ et le mode avancé, appuyez sur F7.
