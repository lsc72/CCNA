Mode avancé de l'UEFI

L'écran du menu principal du mode avancé ASUS s'affiche lorsque vous accédez à ce mode (voir illustration 1). Ce mode fournit des options permettant de configurer les paramètres UEFI avancés.

Remarque : pour revenir au mode EZ, appuyez sur F7.

Voici les éléments figurant dans le menu en haut de l'écran :

  • My Favorites (mes favoris) : (illustration 2) fournit un accès rapide aux paramètres utilisés fréquemment. Appuyez sur F3 ou cliquez sur My Favorites (F3) pour ouvrir l'écran de l'arborescence de la configuration représenté à l'illustration 3.

  • Main (principal) : (illustration 4) permet d'afficher les informations relatives à l'ordinateur et de modifier la configuration de base. Vous pouvez définir les paramètres de date, d'heure, de langue et de sécurité.

  • Ai Tweaker : (illustration 5) permet de modifier les paramètres d'overclocking du processeur et les paramètres de fonctionnement de la mémoire. Usez toujours de prudence lorsque vous modifiez ces paramètres. Des paramètres incorrects peuvent entraîner un dysfonctionnement du système.

  • Advanced (avancé) : (illustration 6) permet de modifier les paramètres du processeur et d'autres périphériques. Usez toujours de prudence lorsque vous modifiez ces paramètres. Des paramètres incorrects peuvent entraîner un dysfonctionnement du système.

  • Moniteur : (illustration 7) permet d'afficher les températures et le statut d'alimentation, et de modifier les paramètres de vitesse du ventilateur.

  • Boot (démarrage) : (illustration 8) permet de modifier les options de démarrage.

  • Tool (outil) : (illustration 9) permet de configurer les options relatives aux fonctions spéciales.

  • Exit (quitter) : (illustration 10) permet de sélectionner les options de sortie et de charger les paramètres par défaut.
