Mise à niveau des périphériques d'entrée et de sortie

Les périphériques doivent être régulièrement mis à niveau. Par exemple, si le périphérique cesse de fonctionner ou si vous souhaitez améliorer les performances et la productivité, une mise à niveau peut s'avérer nécessaire.

Voici quelques raisons de mettre à niveau un clavier et/ou une souris :

  • Remplacer le clavier et la souris par un modèle ergonomique (voir illustration 1). Les périphériques ergonomiques sont conçus pour offrir plus de confort et pour éviter les blessures résultant de mouvements répétitifs.

  • Reconfigurer le clavier pour effectuer une tâche spécifique, telle que la saisie dans une seconde langue avec des caractères supplémentaires.

  • Pour faciliter l'utilisation par les personnes handicapées.

Les écrans peuvent être mis à niveau de différentes façons :

  • Ajoutez un filtre de confidentialité pour empêcher les personnes se trouvant sur le côté de lire les informations affichées à l'écran. Seuls l'utilisateur et les personnes situées juste derrière celui-ci peuvent lire les informations à l'écran.

  • Ajoutez un filtre antireflet pour empêcher la lumière du soleil de se refléter sur l'écran. Un filtre antireflet facilite énormément la lecture à l'écran si celui-ci est exposé à la lumière du jour ou si une source de lumière se situe derrière l'utilisateur.

  • Ajoutez plusieurs écrans (illustration 2), pour améliorer la productivité en permettant à l'utilisateur d'afficher davantage d'informations et de déplacer les données plus facilement et plus rapidement entre les programmes ouverts. Généralement, vous aurez besoin d'une carte vidéo avancée pour prendre en charge les connexions supplémentaires, ou il vous faudra installer une seconde carte vidéo.

Parfois, il n'est pas possible d'effectuer une mise à niveau avec les sockets ou les slots d'extension existants. Si tel est le cas, vous devez faire une mise à niveau à l'aide d'une connexion USB. Si aucun port USB n'est disponible sur l'ordinateur, vous devez installer une carte d'extension USB ou acheter un concentrateur USB (illustration 3).

Lorsque vous installez du nouveau matériel, vous devez éventuellement installer également de nouveaux pilotes. Généralement, vous pouvez le faire en utilisant les supports d'installation. Si vous ne disposez pas des supports d'installation, vous pouvez récupérer les pilotes les plus récents sur le site web du fabricant.

Remarque : un pilote signé est un pilote qui a réussi les tests effectués par le laboratoire WHQL (Windows Hardware Quality Lab) et qui a reçu, de la part de Microsoft, une signature de pilote. L'installation d'un pilote non signé peut entraîner une instabilité du système, des messages d'erreur et des problèmes de démarrage. Si un pilote non signé est détecté pendant l'installation du matériel, vous devrez choisir entre l'arrêt et la poursuite de l'installation. N'installez des pilotes non signés que si leur source est fiable. Certains systèmes d'exploitation Windows n'autorisent pas l'installation de pilotes non signés.
