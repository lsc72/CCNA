Installation du disque dur

Dans un boîtier d'ordinateur, ce sont les baies de disque qui contiennent les disques durs et les lecteurs. Le tableau de l'illustration 1 décrit les trois types les plus courants de baies de disque.

Pour installer un disque dur, trouvez dans le boîtier une baie de disque dur vide dont la taille correspond à celle du disque. Les disques de petit format peuvent souvent être installés dans des baies plus larges au moyen de plateaux ou d'adaptateurs spéciaux.

Pour installer un disque dur 3,5 po (8,9 cm) dans une baie de disque 3,5, procédez comme suit :

Étape 1. Positionnez le disque dur de manière à ce qu'il soit aligné sur la baie de disque.

Étape 2. Insérez le disque dur dans la baie, de façon à ce que les orifices des vis du disque soient alignés sur ceux du boîtier (illustration 2).

Étape 3. Fixez le disque dur dans le boîtier en utilisant les vis appropriées (illustration 3).

Lors de l'installation de plusieurs disques dans un boîtier, il est conseillé de conserver de l'espace entre ceux-ci pour permettre à l'air de circuler et ainsi faciliter le refroidissement. Montez également le lecteur avec la face métallique vers le haut. Cette face métallique permet à la chaleur du disque dur de se dissiper.

Conseil d'installation : serrez légèrement à la main toutes les vis avant de les serrer avec un tournevis. Vous pourrez ainsi serrer plus facilement les deux dernières vis.
