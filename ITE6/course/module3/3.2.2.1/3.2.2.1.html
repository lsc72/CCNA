Informations sur les composants du BIOS

Un technicien peut utiliser les informations du BIOS pour connaître les composants installés dans l'ordinateur, ainsi que certains de leurs attributs. Ces informations peuvent être utiles pour dépanner le matériel qui ne fonctionne pas correctement, mais aussi pour déterminer les options de mise à niveau. Voici quelques informations sur les composants courants qui s'affichent dans le BIOS :

  • Processeur : affiche le fabricant et la vitesse du processeur. Le nombre de processeurs installés apparaît également.

  • Mémoire vive (RAM) : affiche le fabricant et la vitesse de la mémoire vive. Le nombre de slots, ainsi que les slots qu'occupent les modules de mémoire vive, peuvent également être indiqués.

  • Disque dur : affiche le fabricant, la taille et le type des disques durs. Le type et le nombre de contrôleurs de disque dur peuvent également être indiqués.

  • Lecteur optique : affiche le fabricant et le type de lecteurs optiques.
