Configurations du BIOS

Le programme de configuration du BIOS permet également de personnaliser des aspects spécifiques des composants matériels de l'ordinateur, afin de répondre aux besoins de chacun. Les fonctions personnalisables dépendent du fabricant et de la version du BIOS. Avant d'apporter des modifications au BIOS, il est important de bien comprendre leurs répercussions sur l'ordinateur. Des paramètres incorrects peuvent être préjudiciables.

Principaux paramètres de configuration BIOS :

  • Date et heure : la page principale du BIOS comporte un champ System Time (Heure du système) et un champ System Date (Date du système) permettant de régler l'horloge système (illustration 1).

  • Désactivation de périphériques : vous pouvez configurer des paramètres avancés du BIOS pour désactiver les périphériques qui ne sont pas nécessaires ou qui ne sont pas utilisés par l'ordinateur (illustration 2).

  • Ordre de démarrage : juste après le test POST (Power On Self Test), l'ordinateur essaie de charger le système d'exploitation. Le BIOS analyse le premier périphérique dans l'ordre de démarrage à la recherche d'une partition démarrable. La liste détaillant l'ordre de démarrage, ou séquence de démarrage, est une liste ordonnée de périphériques à partir desquels l'ordinateur est autorisé à démarrer. Si le périphérique ne comporte aucune partition démarrable, l'ordinateur analyse le périphérique suivant dans la liste. Lorsque le système détecte un périphérique ayant une partition démarrable, le BIOS recherche le système d'exploitation installé. Cette liste se trouve généralement dans le BIOS sous l'onglet Boot (Démarrage), illustration 3. Vous pouvez indiquer les disques durs, les lecteurs optiques, le démarrage via le réseau, un disque dur USB et la mémoire Flash dans l'ordre de démarrage.

  • Vitesse d'horloge : certains programmes de configuration du BIOS vous permettent de modifier la vitesse d'horloge du processeur, illustration 4. Si vous diminuez la vitesse d'horloge du processeur, l'ordinateur est ralenti et chauffe moins. On parle d'underclocking (sous-cadencement). Le bruit généré par les ventilateurs est ainsi moins important, ce qui peut être utile si vous souhaitez un ordinateur moins bruyant (par exemple, pour un Home cinéma ou dans une chambre). L'overclocking (sur-cadencement) consiste à augmenter la vitesse d'horloge du processeur afin que l'ordinateur fonctionne plus rapidement. Il dégage ainsi davantage de chaleur et est par conséquent plus bruyant en raison de la vitesse plus élevée des ventilateurs.

  • Virtualisation : la technologie de virtualisation permet à un ordinateur d'exécuter plusieurs systèmes d'exploitation dans des fichiers séparés ou des partitions distinctes. Pour ce faire, un programme de virtualisation émule les caractéristiques complètes d'un ordinateur, dont le matériel, le BIOS, le système d'exploitation et les programmes. Activez le paramètre de virtualisation du BIOS si l'ordinateur doit utiliser cette technologie (illustration 5).
