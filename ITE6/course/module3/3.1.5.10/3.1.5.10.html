Installation des câbles externes

Une fois les panneaux du boîtier remis en place, branchez les câbles à l'arrière de l'ordinateur.

REMARQUE : branchez le câble d'alimentation après avoir connecté tous les autres câbles.

Lorsque vous connectez les câbles, assurez-vous qu'ils sont branchés aux bons emplacements sur l'ordinateur. Par exemple, les anciens systèmes utilisent le même type de connecteur PS/2 pour les câbles de souris et de clavier, mais sont différenciés par un code-couleur afin d'éviter tout branchement incorrect. Souvent, une icône du périphérique correspondant (par exemple, un symbole de clavier, souris, écran ou USB) est présente sur le connecteur.

ATTENTION : lorsque vous branchez les câbles, ne forcez jamais.

Pour installer les différents câbles externes, procédez comme suit :

Étape 1. (Illustration 1) Branchez le câble de l'écran sur le port vidéo. Fixez le câble en serrant les vis sur le connecteur. Si l'ordinateur que vous avez assemblé comporte une carte vidéo, veillez à connecter le câble à cette carte, et non au port vidéo de la carte mère.

Étape 2. (Illustration 2) Branchez le câble du clavier sur le port clavier PS/2.

Étape 3. (Illustration 3) Branchez le câble de la souris sur le port souris PS/2.

Étape 4. (Illustration 4) Branchez le câble USB dans un port USB.

Étape 5. (Illustration 5) Branchez le câble d'alimentation dans l'alimentation.

REMARQUE : certaines cartes mères disposent d'un seul port PS/2. Certaines cartes mères ne sont pas équipées de port PS/2 pour la connexion du clavier et de la souris. Connectez un clavier USB, une souris USB ou les deux à ce type de carte mère.

Conseil d'installation : les adaptateurs PS/2 vers USB peuvent être utilisés pour connecter d'anciens périphériques à des cartes mères ne présentant pas de connecteurs PS/2.
