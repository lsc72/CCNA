Protocoles de couche application TCP/IP

Les protocoles d'application TCP/IP spécifient les informations de format et de contrôle nécessaires à un grand nombre de fonctions courantes de communication via Internet. Cliquez sur chaque protocole d'application dans la figure pour en savoir plus sur chacun d'entre eux.

Les protocoles de couche application sont utilisés par les périphériques source et de destination pendant une session de communication. Pour que les communications aboutissent, les protocoles de couche application implémentés sur les hôtes source et de destination doivent être compatibles.

The figure shows examples of OSI layer functionality. Name System (DNS), Host Config (BootP, DHCP), E Mail (SMTP, POP and IMAP), File Transfer (FTP and TFTP) and Web (HTTP). Clicking on any one of them displays a description.
