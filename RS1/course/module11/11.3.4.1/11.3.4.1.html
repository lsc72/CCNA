Utilisation de la commande ipconfig

Comme le montre la figure 1, vous pouvez afficher l'adresse IP de la passerelle par défaut d'un hôte en exécutant la commande ipconfig sur la ligne de commande d'un ordinateur Windows.

Comme illustré à la figure 2, utilisez la commande ipconfig /all pour afficher l'adresse MAC ainsi que les informations sur l'adressage de couche 3 de l'équipement.

Le service Client DNS sur les ordinateurs Windows optimise également les performances de la résolution des noms DNS en stockant en mémoire les noms déjà résolus. Comme le montre la figure 3, la commande ipconfig /displaydns affiche toutes les entrées DNS mises en cache sur un système Windows.

Figure 1 shows the output for the Windows command, “ipconfig”. The IP address, subnet mask, and default gateway are highlighted. Figure 2 shows the output for the Windows command, “ipconfig /all”. The Physical Address or MAC address is highlighted. Figure 3 shows the output for the Windows command, “ipconfig /displaydns”. The DNS information for cisco.com is shown.
