Commande arp

La commande arp est exécutée à partir de l'invite de commande Windows, comme le montre la figure. La commande arp –a fournit la liste de tous les périphériques actuellement présents dans le cache ARP de l'hôte, en précisant leur adresse IPv4, leur adresse physique et leur type d'adressage (statique/dynamique).

Pour mettre à jour les informations stockées dans le cache, l'administrateur réseau peut effacer le cache à l'aide de la commande arp -d*.

Remarque : le cache ARP ne contient que les informations des périphériques auxquels des utilisateurs ont accédé récemment. Pour être sûr que le cache ARP contient des informations sur un périphérique donné dans son tableau ARP, envoyez une requête ping à ce périphérique.
